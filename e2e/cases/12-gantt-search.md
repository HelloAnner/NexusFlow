# 12 甘特图与全局搜索

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-12-001 | 甘特图默认加载 | 打开甘特图页 | 工具栏、分组列、时间轴、状态栏 | `GET /api/gantt`、`GET /api/gantt/summary` |
| E2E-12-002 | 视图维度任务 | 切换任务维度 | 任务/分工层级 | API dimension=task |
| E2E-12-003 | 视图维度人员 | 切换人员维度 | 人员/任务占用 | API dimension=person |
| E2E-12-004 | 视图维度部门 | 切换部门维度 | 部门/人员/任务 | API dimension=department |
| E2E-12-005 | 视图维度项目 | 切换项目维度 | 项目/任务/分工 | API dimension=project |
| E2E-12-006 | 时间粒度切换 | 切日、周、月、季度 | 四种时间轴 | API granularity 匹配 |
| E2E-12-007 | 日期范围选择 | 选择自定义日期 | 时间轴变化 | API start/end 匹配 |
| E2E-12-008 | 只看风险 | 打开风险开关 | 只显示风险条目 | API risk filter |
| E2E-12-009 | 隐藏已完成 | 打开隐藏已完成 | 完成条目消失 | API status filter |
| E2E-12-010 | 刷新甘特 | 点击刷新 | 数据更新时间变化 | API 重新请求 |
| E2E-12-011 | 导出甘特 | 点击导出 | 导出结果 | 导出接口或文件生成 |
| E2E-12-012 | 筛选抽屉打开 | 点击筛选 | 抽屉 | 可选项 API/页面一致 |
| E2E-12-013 | 甘特按任务类型筛选 | 选择任务类型 | 结果 | API filter 匹配 |
| E2E-12-014 | 甘特按状态筛选 | 选择状态 | 结果 | API filter 匹配 |
| E2E-12-015 | 甘特按项目筛选 | 选择项目 | 结果 | API filter 匹配 |
| E2E-12-016 | 甘特按组织筛选 | 选择组织 | 结果 | API filter 匹配 |
| E2E-12-017 | 甘特按负责人/成员筛选 | 选择人员 | 结果 | API filter 匹配 |
| E2E-12-018 | 甘特按风险等级筛选 | 选择风险 | 结果 | API filter 匹配 |
| E2E-12-019 | 甘特按隐藏项目筛选 | 授权用户筛隐藏项目 | 结果 | 未授权无筛选值 |
| E2E-12-020 | 甘特跨部门/待审批筛选 | 选择跨部门、待审批 | 结果 | approval/dispatch 数据匹配 |
| E2E-12-021 | 筛选无结果空态 | 设置无结果条件 | 空态、清除筛选 | API 返回空 items |
| E2E-12-022 | 保存个人筛选 | 保存筛选为视图 | 保存成功 | `POST /api/saved-filters` |
| E2E-12-023 | 加载保存筛选 | 重新进入选择保存视图 | 条件恢复 | `GET /api/saved-filters` |
| E2E-12-024 | 甘特条目悬浮浮层 | 悬浮任务/项目/分工/人员状态条 | 标题、类型、负责人、成员、日期、进度、风险、操作 | item 字段一致 |
| E2E-12-025 | 点击项目条 | 点击项目条 | 项目详情 | project API |
| E2E-12-026 | 点击任务条 | 点击任务条 | 任务详情 | task API |
| E2E-12-027 | 点击分工条 | 点击分工条 | 任务详情分工页签 | assignment 字段 |
| E2E-12-028 | 点击人员状态条 | 点击人员状态条 | 人员负载页签 | workload API |
| E2E-12-029 | 点击冲突标记 | 点击冲突标记 | 冲突详情 | conflict API |
| E2E-12-030 | 无详情权限显示已占用 | 受限用户看人员负载 | “已占用”脱敏 | API 不返回任务名称 |
| E2E-12-031 | 隐藏项目未授权完全不显示 | 未授权用户看甘特 | 无名称、摘要、时间 | API 不返回隐藏对象 |
| E2E-12-032 | 全局搜索弹窗打开 | 点击顶部搜索或快捷键 | 最近访问、最近搜索、分类结果 | `GET /api/search/suggest` |
| E2E-12-033 | 搜索任务即时结果 | 输入任务名 | 任务结果 | suggest/search 返回 task |
| E2E-12-034 | 搜索项目即时结果 | 输入项目名 | 项目结果 | 返回 project |
| E2E-12-035 | 搜索人员即时结果 | 输入人员名 | 人员结果 | 返回 person |
| E2E-12-036 | 搜索资料即时结果 | 输入资料名 | 资料结果 | 返回 resource |
| E2E-12-037 | 搜索结果页 | 提交搜索 | 搜索结果页 | `GET /api/search?q=...` |
| E2E-12-038 | 搜索结果按类型筛选 | 筛任务、项目、人员、资料 | 结果 | type 匹配 |
| E2E-12-039 | 搜索结果按状态/组织/项目/时间筛选 | 逐项筛选 | 结果 | filter 匹配 |
| E2E-12-040 | 搜索结果打开详情 | 点击结果 | 对应详情 | object API |
| E2E-12-041 | 搜索权限裁剪 | 未授权用户搜索隐藏对象关键词 | 无结果或脱敏 | API 不泄露 |
| E2E-12-042 | 移动端甘特轻量视图 | 移动视口打开甘特 | 列表时间线、横向滚动、风险跳转 | API 一致 |
| E2E-12-043 | 移动端全屏搜索 | 移动视口打开搜索 | 全屏搜索、分类结果 | search API |

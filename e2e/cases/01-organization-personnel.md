# 01 组织与人员

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-01-001 | 组织管理页加载组织树 | 登录 SA；进入组织管理页 | 组织树、默认详情 | `GET /api/orgs/tree` 返回层级数据 |
| E2E-01-002 | 点击组织节点切换详情 | 点击中心、部门、工作室节点 | 每个节点详情 | `GET /api/orgs` 含对应组织 |
| E2E-01-003 | 新增同级组织 | 在组织节点旁新增同级组织 | 表单、成功提示、树更新 | `POST /api/orgs` 成功；列表可读回 |
| E2E-01-004 | 新增下级组织 | 选择部门新增工作室 | 表单、树层级更新 | 新组织 `parent_id/path` 正确 |
| E2E-01-005 | 编辑组织负责人和默认审批人 | 打开组织编辑；选择负责人、副负责人、默认审批人 | 编辑前、保存后 | `PATCH /api/orgs/{id}` 成功；审计有记录 |
| E2E-01-006 | 停用空组织 | 停用无人员无下级组织 | 影响确认、停用后状态 | `POST /api/orgs/{id}/disable` 成功 |
| E2E-01-007 | 有人员组织禁止直接删除或停用需提示 | 尝试停用含人员组织 | 阻断或影响范围提示 | API 返回 400/409 或确认后符合规则 |
| E2E-01-008 | 组织负责人为空时审批回退 | 清空部门负责人；创建跨部门派发 | 审批人展示 | 协调单审批人来自上级默认审批人 |
| E2E-01-009 | 人员列表默认加载 | 进入人员列表页 | 列表、分页、负载颜色 | `GET /api/users?page_size=...` 返回 `items` |
| E2E-01-010 | 人员按组织筛选 | 选择组织筛选 | 筛选条件、结果 | API 查询结果均属于该组织 |
| E2E-01-011 | 人员按技能筛选 | 选择技能标签 | 筛选条件、结果 | `GET /api/skills` 有标签；人员包含标签 |
| E2E-01-012 | 人员按项目归属筛选 | 选择项目 | 结果列表 | 返回人员均有关联项目 |
| E2E-01-013 | 人员按工作状态筛选 | 选择在岗、休假、出差 | 每种筛选结果 | API 返回 `work_status` 匹配 |
| E2E-01-014 | 人员按账号状态筛选 | 选择待审核、启用、禁用、锁定 | 每种筛选结果 | API 返回账号状态匹配 |
| E2E-01-015 | 人员按负载范围筛选 | 选择 0-60、60-90、90-100、超载 | 负载颜色和人数 | `GET /api/users/{id}/workload-summary` 与颜色一致 |
| E2E-01-016 | 人员详情基础信息页签 | 点击人员姓名进入详情 | 顶部信息、基础信息 | `GET /api/users/{id}` 返回基础字段 |
| E2E-01-017 | 人员详情项目归属页签 | 打开项目归属 | 项目列表 | 项目成员 API 或人员详情项目字段一致 |
| E2E-01-018 | 人员详情技能标签页签 | 打开技能标签并编辑 | 编辑前后 | `PATCH /api/users/{id}` 成功；技能可读回 |
| E2E-01-019 | 人员详情当前任务页签 | 打开当前任务 | 任务列表 | `GET /api/tasks?member=...` 或人员任务数据匹配 |
| E2E-01-020 | 人员详情负载日历 | 切换周/月；点击日期 | 周/月视图、日期明细 | `GET /api/workload/person/{id}` 返回日期负载 |
| E2E-01-021 | 人员详情历史记录 | 打开历史记录 | 组织、角色、状态变更 | 审计或事件包含变更 |
| E2E-01-022 | 新建人员成功 | 填姓名、账号、主组织、角色 | 表单、成功后列表 | `POST /api/users` 成功；用户可读回 |
| E2E-01-023 | 新建人员必填校验 | 清空姓名、账号、主组织、角色提交 | 字段错误 | API 返回 400 |
| E2E-01-024 | 重复账号禁止保存 | 使用已有账号新建 | 错误提示 | API 返回 409 或 400 |
| E2E-01-025 | 离职人员不可设为负责人 | 将人员设为离职后在任务选择负责人 | 阻断提示 | 创建任务或预览 API 返回校验错误 |
| E2E-01-026 | 待审核人员不可派发 | 使用待审核人员作为负责人 | 阻断提示 | API 返回业务错误 |
| E2E-01-027 | 非 SA 不能改账号状态 | 普通管理员进入人员编辑 | 字段隐藏或禁用 | `PATCH /api/users/{id}` 返回 403 |
| E2E-01-028 | 待审核注册入口展示 | 人员模块打开待审核入口 | 待审核列表 | `GET /api/admin/registrations` 返回待审核数据 |
| E2E-01-029 | 审核入口通过注册 | 点击通过 | 资料、确认、成功 | 审核 API 成功；用户状态更新 |
| E2E-01-030 | 审核入口拒绝注册 | 点击拒绝并填写原因 | 原因表单、结果 | 审核 API 成功；申请状态 rejected |
| E2E-01-031 | 修改组织后通过注册 | 审核时调整组织 | 调整前后 | 用户主组织为最终选择 |
| E2E-01-032 | 修改角色后通过注册 | 审核时调整角色 | 调整前后 | 用户角色为最终选择 |
| E2E-01-033 | 合并到已有人员 | 选择已有人员合并 | 合并确认、结果 | 注册申请关联已有人员 |
| E2E-01-034 | 技能标签管理新增标签 | 进入技能管理；新增分组/标签 | 新增表单、列表 | `POST /api/skills` 成功 |
| E2E-01-035 | 技能标签重名校验 | 同分组新增同名标签 | 错误提示 | API 返回 409/400 |
| E2E-01-036 | 停用技能标签 | 停用已存在标签 | 停用确认、历史展示 | `POST /api/skills/{id}/disable` 成功；不可新选择 |
| E2E-01-037 | 被使用标签不可删除 | 尝试删除被使用标签 | 阻断提示 | API 返回 409/400 |

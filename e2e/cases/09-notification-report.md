# 09 通知、待办与报表

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-09-001 | 待办中心加载 | 打开待办中心 | 待办列表 | `GET /api/todos` 返回 items |
| E2E-09-002 | 待办按类型筛选 | 审批、进度、验收、资料、系统逐项筛选 | 每类结果 | type 匹配 |
| E2E-09-003 | 待办按状态筛选 | 未完成、已完成、已读、未读 | 结果 | status/read 匹配 |
| E2E-09-004 | 待办按风险筛选 | 低、中、高、严重 | 结果 | risk_level 匹配 |
| E2E-09-005 | 待办按任务和发起人筛选 | 选择任务/发起人 | 结果 | object/creator 匹配 |
| E2E-09-006 | 待办按截止时间筛选 | 本周/逾期/自定义 | 结果 | due_at 匹配 |
| E2E-09-007 | 审批待办处理跳协调单 | 点击立即处理 | 协调单详情 | approval API 成功 |
| E2E-09-008 | 进度待办处理跳分工填报 | 点击处理 | 分工进度表单 | assignment API 成功 |
| E2E-09-009 | 验收待办处理跳任务验收 | 点击处理 | 验收页 | task detail/acceptance API |
| E2E-09-010 | 资料待办处理跳资料页签 | 点击处理 | 任务资料页签 | resources check |
| E2E-09-011 | 待办标记已读/完成 | 点击标记 | 状态变化 | `POST /api/todos/{id}/complete` |
| E2E-09-012 | 待办转交权限控制 | 普通用户/管理员分别转交 | 有权可转、无权禁止 | API 权限一致 |
| E2E-09-013 | 查看待办来源 | 点击来源 | 来源详情页 | object API 成功 |
| E2E-09-014 | 通知中心加载 | 打开通知中心 | 通知列表 | `GET /api/notifications` |
| E2E-09-015 | 通知分类筛选 | 任务、审批、风险、资料、系统 | 结果 | category 匹配 |
| E2E-09-016 | 通知已读 | 点击通知或标记已读 | 已读状态 | `POST /api/notifications/{id}/read` |
| E2E-09-017 | 通知来源跳转 | 点击来源对象 | 详情页 | object API 成功 |
| E2E-09-018 | 报表首页加载 | 打开报表首页 | 指标卡、图表 | `GET /api/reports` |
| E2E-09-019 | 报表首页指标口径 | 核对进行中、高风险、本周到期、待审批、平均负载、资料完整率 | 指标截图 | 对应任务/冲突/审批/资料 API 聚合一致 |
| E2E-09-020 | 任务状态分布图 | 查看图表 | 图表 | report 数据状态分布 |
| E2E-09-021 | 任务类型分布图 | 查看图表 | 图表 | report 数据类型分布 |
| E2E-09-022 | 部门负载排行 | 查看排行 | 图表 | workload/report 一致 |
| E2E-09-023 | 重点任务进度 | 查看图表 | 图表 | tasks 重点任务进度一致 |
| E2E-09-024 | 任务总览报表筛选 | 按组织、项目、类型、时间、负责人、状态筛选 | 筛选结果 | `GET /api/reports/task-overview` |
| E2E-09-025 | 延期任务列表跳转 | 点击延期任务 | 任务详情 | task API 成功 |
| E2E-09-026 | 人员负载报表 | 打开负载报表 | 人员、部门、工时、负载率、超载天数 | `GET /api/reports/workload` |
| E2E-09-027 | 负载报表跳人员详情 | 点击人员 | 人员详情负载页签 | workload API |
| E2E-09-028 | 资料归档报表 | 打开资料归档报表 | 任务、必需资料、已确认、缺失、完整率 | `GET /api/reports/resource-archive` |
| E2E-09-029 | 资料报表跳补充资料 | 点击缺失资料任务 | 任务资料页签 | check-requirements |
| E2E-09-030 | 协调统计报表 | 打开协调统计 | 协调类型、状态、耗时、超时 | approval/report API |
| E2E-09-031 | 报表导出 | 点击导出 | 导出结果 | `POST /api/reports/{type}/export` |
| E2E-09-032 | 报表权限裁剪 | 不同角色打开同一报表 | 指标差异 | report API 按范围裁剪 |

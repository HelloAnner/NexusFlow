# 05 负载与冲突

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-05-001 | 人员负载周视图 | 打开人员负载日历周视图 | 周日历 | `GET /api/workload/person/{id}` 日期负载一致 |
| E2E-05-002 | 人员负载月视图 | 切换月视图 | 月日历 | API 返回整月数据 |
| E2E-05-003 | 自定义范围负载 | 选择自定义日期范围 | 日历范围 | API start/end 匹配 |
| E2E-05-004 | 低负载绿色 | 找到低于 60% 日期 | 绿色格子 | API load < 60 |
| E2E-05-005 | 中负载黄色 | 找到 60%-90% 日期 | 黄色格子 | API load 区间匹配 |
| E2E-05-006 | 高负载橙色 | 找到 90%-100% 日期 | 橙色格子 | API load 区间匹配 |
| E2E-05-007 | 超载红色 | 找到超过 100% 日期 | 红色格子 | API load > 100 |
| E2E-05-008 | 全天占用灰色 | 找到全天任务日期 | 灰色块 | API all_day=true |
| E2E-05-009 | 点击日期展示明细 | 点击某日 | 任务、分工、投入、负责人、状态、重点标识 | 明细与 API 一致 |
| E2E-05-010 | 负载预估 | 新建任务人员选择阶段预估 | 预估结果 | `POST /api/workload/preview` 返回每日负载 |
| E2E-05-011 | 冲突中心统计 | 打开冲突中心 | 未处理、高风险、今日新增、超时统计 | `GET /api/conflicts` 聚合一致 |
| E2E-05-012 | 冲突类型筛选 | 按工时超载、全天重叠、休假冲突、跨部门未审批筛选 | 每类结果 | API conflict_type 匹配 |
| E2E-05-013 | 风险等级筛选 | 按低、中、高、严重筛选 | 筛选结果 | risk_level 匹配 |
| E2E-05-014 | 处理状态筛选 | 按未处理、处理中、已解决、强制安排筛选 | 筛选结果 | status 匹配 |
| E2E-05-015 | 涉及人员/部门/任务筛选 | 逐项筛选 | 筛选条件和结果 | API 结果匹配 |
| E2E-05-016 | 冲突详情基础信息 | 打开冲突详情抽屉 | 编号、类型、等级、来源、生成时间 | `GET /api/conflicts/{id}` 一致 |
| E2E-05-017 | 冲突详情时间线 | 查看日期时间线 | 已有任务、新任务、投入、超载 | 详情 API 一致 |
| E2E-05-018 | 工时超载建议 | 打开工时超载冲突 | 降低投入/调整日期建议 | recommendation 匹配 |
| E2E-05-019 | 全天重叠建议 | 打开全天重叠冲突 | 替换人员/强制安排建议 | recommendation 匹配 |
| E2E-05-020 | 休假冲突建议 | 打开休假冲突 | 调整周期/取消派发建议 | recommendation 匹配 |
| E2E-05-021 | 跨部门未审批建议 | 打开跨部门冲突 | 发起协调建议 | recommendation 匹配 |
| E2E-05-022 | 修改任务计划处理冲突 | 从冲突详情进入修改任务 | 修改页、处理后 | 冲突状态更新或重新计算 |
| E2E-05-023 | 修改分工投入处理冲突 | 修改投入小时 | 修改前后 | assignment 更新，冲突缓解 |
| E2E-05-024 | 发起协调处理冲突 | 点击发起协调 | 协调单创建 | approval 创建 |
| E2E-05-025 | 标记已解决 | 点击已解决并填说明 | 说明、状态 | `POST /api/conflicts/{id}/resolve` 成功 |
| E2E-05-026 | 强制安排 | 点击强制安排并填原因 | 二次确认、状态 | `POST /api/conflicts/{id}/force` 成功且审计 |
| E2E-05-027 | 冲突重算 | 点击重算 | 重算前后列表 | `POST /api/conflicts/recalculate` 成功 |
| E2E-05-028 | 派发页冲突检查阻断 | 高风险任务发布 | 冲突检查阻断 | submit 返回阻断 |
| E2E-05-029 | 甘特图风险标记 | 打开甘特图查看冲突任务 | 红色底纹、描边、延期标记、悬浮摘要 | gantt 返回 risk/conflict 字段 |

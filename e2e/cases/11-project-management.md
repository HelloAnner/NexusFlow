# 11 项目管理

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-11-001 | 项目列表加载 | 打开项目列表 | 项目表格 | `GET /api/projects` |
| E2E-11-002 | 按名称编号筛选 | 搜索项目名/编号 | 结果 | query 匹配 |
| E2E-11-003 | 按类型级别状态筛选 | 分别筛类型、级别、状态 | 结果 | 字段匹配 |
| E2E-11-004 | 按组织负责人筛选 | 选择组织/负责人 | 结果 | org/owner 匹配 |
| E2E-11-005 | 隐藏项目筛选 | SA 筛隐藏项目 | 结果 | visibility=hidden |
| E2E-11-006 | 风险等级和时间筛选 | 筛风险、日期范围 | 结果 | risk/date 匹配 |
| E2E-11-007 | 新建项目 | 填名称、编号、类型、级别、负责人、组织、日期、概述、隐藏标记 | 表单、结果 | `POST /api/projects` 成功 |
| E2E-11-008 | 编辑项目 | 修改基础字段 | 编辑前后 | `PATCH /api/projects/{id}` |
| E2E-11-009 | 添加成员入口 | 列表点击添加成员 | 成员弹窗 | members API 可用 |
| E2E-11-010 | 查看项目甘特入口 | 列表点击甘特 | 项目甘特 | gantt project filter |
| E2E-11-011 | 项目归档 | 已完成项目点击归档 | 确认、归档状态 | `POST /api/projects/{id}/archive` |
| E2E-11-012 | 管理授权入口 | 隐藏项目点击授权 | 授权页 | visibility grants |
| E2E-11-013 | 操作按钮按权限显示 | 不同角色打开项目列表 | 按钮差异 | 对应写 API 403/200 |
| E2E-11-014 | 项目详情顶部信息 | 打开项目详情 | 名称、编号、状态、可见性、负责人、组织、日期、风险、主操作 | `GET /api/projects/{id}` |
| E2E-11-015 | 项目概览页签 | 查看概览 | 概述、成果、指标、重点任务、高风险、动态 | `GET /api/projects/{id}/stats` |
| E2E-11-016 | 项目关键指标口径 | 核对总任务、进行中、延期、本周工时、成员数、资料完整率 | 指标截图 | stats 与 tasks/resources 一致 |
| E2E-11-017 | 成员页签列表 | 打开成员页签 | 姓名、组织、身份、负责工作、任务数、负载、加入、有效 | project members API |
| E2E-11-018 | 添加项目成员 | 添加成员和身份 | 添加前后 | `POST /api/projects/{id}/members` |
| E2E-11-019 | 修改成员身份 | 修改身份/负责工作 | 修改前后 | `PATCH /api/projects/{id}/members/{person_id}` |
| E2E-11-020 | 标记成员退出 | 退出项目 | 确认、状态 | DELETE 或 PATCH 成员状态 |
| E2E-11-021 | 添加项目成员不占工时 | 添加成员后看负载 | 负载未变 | workload API 不变 |
| E2E-11-022 | 外部门成员提示审批 | 添加外部门成员 | 提示 | 后续任务派发需审批 |
| E2E-11-023 | 隐藏项目成员仅授权可见 | 未授权/授权用户查看成员页 | 隐藏/可见对比 | API 裁剪 |
| E2E-11-024 | 项目任务页签 | 打开任务页签 | 任务列表和筛选 | `GET /api/tasks?project_id=...` |
| E2E-11-025 | 新建项目任务 | 项目详情点击新建任务 | 项目预填 | 新任务 project_id 正确 |
| E2E-11-026 | 项目任务导出 | 导出有权限任务清单 | 导出结果 | 导出 API 权限裁剪 |
| E2E-11-027 | 项目甘特页签 | 打开甘特 | 项目周期、任务、分工、里程碑、风险、负载 | gantt API project 过滤 |
| E2E-11-028 | 项目甘特点击任务 | 点击任务条 | 任务详情 | task API 成功 |
| E2E-11-029 | 项目甘特点击人员 | 点击成员行 | 人员详情 | user API 成功 |
| E2E-11-030 | 项目甘特点击冲突 | 点击冲突标记 | 冲突详情 | conflict API 成功 |
| E2E-11-031 | 项目甘特维度切换 | 按任务、成员、阶段切换 | 三种视图 | gantt 参数匹配 |
| E2E-11-032 | 项目资料页签 | 打开资料 | 项目资料、任务资料、阶段、最终、归档 | resources project filter |
| E2E-11-033 | 项目资料上传 | 上传项目资料 | 上传前后 | resources API |
| E2E-11-034 | 关联任务资料 | 关联已有任务资料 | 关联结果 | link API |
| E2E-11-035 | 项目授权页签 | 打开授权 | 授权对象、范围、生效、失效、授权人、原因 | visibility grants |
| E2E-11-036 | 新增项目授权 | 新增授权 | 授权表单、结果 | `POST /api/projects/{id}/visibility-grants` |
| E2E-11-037 | 修改授权范围 | 调整授权范围 | 修改前后 | grant 更新 |
| E2E-11-038 | 撤销授权 | 撤销并确认 | 授权消失 | 未授权用户不可见 |
| E2E-11-039 | 授权日志 | 查看授权日志 | 操作人、时间、摘要、来源 | audit API |
| E2E-11-040 | 项目日志页签 | 打开日志 | 创建、信息变更、成员、授权、状态、归档 | event/audit 一致 |

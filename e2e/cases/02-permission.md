# 02 权限

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-02-001 | 角色管理页加载 | SA 进入角色管理 | 角色列表 | `GET /api/roles` 返回角色 |
| E2E-02-002 | 新增角色 | 点击新增角色，填写类型、数据范围、派发范围 | 表单、成功后列表 | `POST /api/roles` 成功 |
| E2E-02-003 | 复制角色 | 复制已有角色并改名 | 复制前、复制后 | 新角色 actions 与源角色一致 |
| E2E-02-004 | 编辑角色基础信息 | 修改角色名、启用状态 | 编辑弹窗、结果 | `PATCH /api/roles/{id}` 成功 |
| E2E-02-005 | 停用角色 | 停用非内置角色 | 确认、列表状态 | 角色不可再被分配 |
| E2E-02-006 | 中心主任/书记不可删除 | 尝试删除或停用内置角色 | 阻断提示 | API 返回 400/403 |
| E2E-02-007 | 至少保留一个系统管理员 | 停用最后一个系统管理员角色 | 风险提示 | API 阻断 |
| E2E-02-008 | 角色详情基础页签 | 打开角色详情 | 基础信息 | `GET /api/roles/{id}/actions` 可访问 |
| E2E-02-009 | 菜单权限配置 | 勾选和取消菜单 | 权限矩阵变化 | 保存后 `permissions/me` 菜单变化 |
| E2E-02-010 | 操作权限矩阵保存 | 勾选任务查看、新建、编辑、派发、审批、验收、归档、导出 | 矩阵、保存结果 | `PUT /api/roles/{id}/actions` 后读回一致 |
| E2E-02-011 | 高风险权限二次确认 | 开启全中心导出、隐藏授权等 | 二次确认 | 审计记录风险级别 |
| E2E-02-012 | 权限变更必须填写原因 | 不填原因保存高风险变更 | 错误提示 | API 返回 400 |
| E2E-02-013 | 授权人员页签 | 查看角色授权人员 | 人员列表 | 用户 role_ids 包含该角色 |
| E2E-02-014 | 变更记录页签 | 修改角色后查看记录 | 变更前后摘要 | `GET /api/audit/permission` 有记录 |
| E2E-02-015 | 数据范围配置人员范围 | 配置本人、本部门、分管部门、全中心 | 每种配置截图 | 普通账号 `GET /api/users` 裁剪正确 |
| E2E-02-016 | 数据范围配置任务范围 | 配置任务范围 | 配置页、任务列表 | `GET /api/tasks` 只返回授权任务 |
| E2E-02-017 | 数据范围配置项目范围 | 配置项目范围 | 配置页、项目列表 | `GET /api/projects` 只返回授权项目 |
| E2E-02-018 | 数据范围配置资料范围 | 配置资料范围 | 配置页、资料列表 | `GET /api/resources` 裁剪正确 |
| E2E-02-019 | 数据范围配置报表范围 | 配置报表范围 | 报表数据变化 | `GET /api/reports/{type}` 指标按范围变化 |
| E2E-02-020 | 数据范围配置甘特范围 | 配置甘特范围 | 甘特视图变化 | `GET /api/gantt` 条目裁剪 |
| E2E-02-021 | 自定义组织范围 | 指定组织集合 | 配置和用户视角 | 用户只能看到指定组织数据 |
| E2E-02-022 | 自定义项目范围 | 指定项目集合 | 配置和用户视角 | 用户只能看到指定项目数据 |
| E2E-02-023 | 自定义人员范围 | 指定人员集合 | 配置和用户视角 | 用户只能看到指定人员 |
| E2E-02-024 | 隐藏项目授权给人员 | 为隐藏项目授权人员查看任务 | 授权前不可见、授权后可见 | `POST /api/visibility-grants` 后目标用户可见 |
| E2E-02-025 | 隐藏项目授权给角色 | 授权角色 | 授权页、角色用户视角 | 角色用户可见 |
| E2E-02-026 | 隐藏项目授权给组织 | 授权组织 | 授权页、组织用户视角 | 组织用户可见 |
| E2E-02-027 | 隐藏项目授权项目成员 | 授权项目成员 | 授权页、成员视角 | 成员可见 |
| E2E-02-028 | 授权范围分别验证 | 查看任务、查看资料、编辑任务、参与分工、审批、验收逐项授权 | 每个范围下页面按钮 | `permissions/check` 和实际 API 权限一致 |
| E2E-02-029 | 授权必须有开始时间 | 新增授权不填开始时间 | 字段错误 | API 返回 400 |
| E2E-02-030 | 授权结束时间到期失效 | 设置短期授权并过期 | 到期前后可见性 | 到期后对象不可见 |
| E2E-02-031 | 取消授权即时生效 | 删除授权后刷新目标用户页面 | 授权前后 | `DELETE /api/visibility-grants/{id}` 后不可见 |
| E2E-02-032 | 取消授权保留历史日志 | 删除授权后看日志 | 日志详情 | 审计记录仍可查 |
| E2E-02-033 | 权限审计按操作人筛选 | 选择操作人 | 筛选结果 | `GET /api/audit/permission` 结果匹配 |
| E2E-02-034 | 权限审计按对象筛选 | 选择对象 | 筛选结果 | 结果 object_id 匹配 |
| E2E-02-035 | 权限审计详情弹窗 | 打开详情 | before/after JSON | 响应包含变更前后 |
| E2E-02-036 | 待审核用户业务数据隔离 | 待审核用户登录 | 仅审核状态/基础入口 | 业务 API 返回 403 |

# 06 资料库

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-06-001 | 资料库列表加载 | 打开资料库 | 列表、分页 | `GET /api/resources` 返回 items |
| E2E-06-002 | 按资料名称筛选 | 输入名称搜索 | 结果 | API 查询匹配 |
| E2E-06-003 | 按资料类型筛选 | 选择报告/PPT/数据等 | 结果 | resource_type 匹配 |
| E2E-06-004 | 按关联任务筛选 | 选择任务 | 结果 | resource_link 含任务 |
| E2E-06-005 | 按关联项目筛选 | 选择项目 | 结果 | 关联项目匹配 |
| E2E-06-006 | 按上传人和组织筛选 | 选择上传人/组织 | 结果 | uploader/org 匹配 |
| E2E-06-007 | 按状态筛选 | 已提交、已确认、退回、归档 | 结果 | status 匹配 |
| E2E-06-008 | 阶段成果筛选 | 只看阶段成果 | 结果 | stage flag 匹配 |
| E2E-06-009 | 最终成果筛选 | 只看最终成果 | 结果 | final flag 匹配 |
| E2E-06-010 | 上传资料成功 | 打开上传弹窗，选择类型、文件、任务、分工、版本说明、成果标记 | 上传表单、上传后列表 | upload-url、complete-upload 成功 |
| E2E-06-011 | 上传资料类型必填 | 不选类型提交 | 字段错误 | API 返回 400 |
| E2E-06-012 | 上传文件必填 | 不选文件提交 | 字段错误 | API 返回 400 |
| E2E-06-013 | 无任务权限禁止上传 | 未授权用户给任务上传 | 权限提示 | complete-upload 返回 403 |
| E2E-06-014 | 最终成果权限校验 | 非负责人/验收人标最终成果 | 阻断提示 | API 返回 403 |
| E2E-06-015 | 下载资料 | 点击下载 | 下载提示 | `GET /api/resources/{id}/download-url` 返回 URL |
| E2E-06-016 | 查看资料详情 | 点击资料名 | 顶部信息、预览页签 | `GET /api/resources/{id}` 字段一致 |
| E2E-06-017 | 版本历史 | 打开版本历史 | 版本列表 | 详情 versions 一致 |
| E2E-06-018 | 上传新版本 | 在资料详情上传新版本 | 新版本表单、版本号变化 | `POST /api/resources/{id}/versions` 成功 |
| E2E-06-019 | 关联任务 | 为资料关联任务/分工 | 关联结果 | `POST /api/resources/{id}/link` 成功 |
| E2E-06-020 | 标记阶段成果 | 点击阶段成果 | 标记变化 | 资源字段更新 |
| E2E-06-021 | 标记最终成果 | 负责人标最终成果 | 标记变化 | 资源字段更新 |
| E2E-06-022 | 确认资料 | 验收人确认资料 | 状态变更 | 状态 confirmed |
| E2E-06-023 | 退回资料 | 验收人退回并填原因 | 原因、状态 | 状态 rejected；待办生成 |
| E2E-06-024 | 归档资料 | 归档资料 | 归档确认 | `POST /api/resources/{id}/archive` 成功 |
| E2E-06-025 | 任务资料页签必需资料清单 | 打开任务详情资料页签 | 必需资料、缺失提示、上传入口 | `GET /api/resources/check-requirements` 一致 |
| E2E-06-026 | 任务资料阶段成果区 | 上传阶段成果后查看 | 阶段成果区 | 资源关联正确 |
| E2E-06-027 | 任务资料最终成果区 | 上传最终成果后查看 | 最终成果区 | 资源关联正确 |
| E2E-06-028 | 验收前资料完整性阻断 | 缺必需资料提交验收 | 阻断提示 | check-requirements 返回 missing |
| E2E-06-029 | 隐藏项目资料未授权不可见 | 未授权用户搜索资料 | 空态 | API 不返回资料 |

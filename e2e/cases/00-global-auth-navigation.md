# 00 全局、登录、导航

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-00-001 | 未登录访问首页被重定向登录 | 清空浏览器 storage；打开 `/` | 登录页、URL | `GET /api/auth/me` 返回 401 |
| E2E-00-002 | SA 账号密码登录成功 | 打开 `/login`；输入 `Anner/1`；提交 | 登录前、登录后首页 | `POST /api/auth/login` 返回 token；`GET /api/auth/me` 返回 `login_name=Anner` |
| E2E-00-003 | 错误密码登录失败 | 输入正确账号错误密码；提交 | 错误提示 | 登录接口返回 401；失败次数递增或错误码稳定 |
| E2E-00-004 | 账号禁用后登录失败 | 用 SA 禁用测试账号；用该账号登录 | 禁用提示 | 登录接口返回 403；账号状态为 disabled |
| E2E-00-005 | 账号锁定后登录失败 | 连续错误登录至锁定阈值 | 锁定提示 | 登录接口返回 403；账号状态为 locked |
| E2E-00-006 | 退出登录清除会话 | 已登录点击头像退出 | 退出前、登录页 | `POST /api/auth/logout` 成功；旧 token 调 `auth/me` 返回 401 |
| E2E-00-007 | 全局导航逐项可达 | 登录后依次点击首页、任务、甘特、项目、人员、冲突、资料、工具、配置、系统管理 | 每个页面首屏 | 对应页面首屏 API 返回 200 或按权限 403 |
| E2E-00-008 | 全局新建任务入口 | 点击顶部“新建任务” | 新建任务页 | `GET /api/users`、`GET /api/projects` 返回可选项 |
| E2E-00-009 | 全局搜索任务跳转 | 输入已存在任务名搜索并提交 | 搜索前、任务列表查询结果 | `GET /api/tasks?q=...` 返回匹配任务 |
| E2E-00-010 | 404 路由回首页 | 打开不存在路径 `/unknown-path` | 回到首页 | 静态资源返回前端应用且无 500 |
| E2E-00-011 | 登录态刷新保持 | 登录后刷新页面 | 刷新前、刷新后 | localStorage token 存在；`auth/me` 成功 |
| E2E-00-012 | token 失效后自动回登录 | 删除或伪造 token 后刷新受保护页面 | 被踢回登录 | 业务 API 返回 401 |
| E2E-00-013 | 健康检查 | 打开或 curl `/healthz`、`/readyz` | readyz JSON 截图或保存响应 | status、database、port、search_backend 字段存在 |
| E2E-00-014 | 移动端导航与内容不重叠 | 设置移动视口；登录；打开主要页面 | 首页、任务、人员、详情页移动截图 | API 与桌面一致 |
| E2E-00-015 | 页面加载态和异常态 | 拦截业务接口为慢响应和 500 | 加载态、错误态 | 错误状态不应导致白屏 |

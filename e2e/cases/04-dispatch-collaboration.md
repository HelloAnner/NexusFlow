# 04 派发协同

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-04-001 | 派发向导加载任务模板 | 进入新建任务派发向导，选择任务类型 | 模板字段 | 配置 API 返回对应模板 |
| E2E-04-002 | 发起人角色预设组织 | 不同角色进入向导 | 组织默认值 | 提交 payload 组织正确 |
| E2E-04-003 | 项目决定候选成员范围 | 选择项目后查看负责人/成员候选 | 候选列表 | 候选人来自项目成员或授权范围 |
| E2E-04-004 | 负责人权限校验 | 选择不可见任务的人员为负责人 | 阻断提示 | `dispatch/preview` 返回不可担任 |
| E2E-04-005 | 离职负责人校验 | 选择离职人员 | 阻断提示 | API 返回业务错误 |
| E2E-04-006 | 负责人休假风险提示 | 选择休假人员 | 风险提示 | preview 返回状态冲突 |
| E2E-04-007 | 成员批量导入 | 上传或粘贴成员清单 | 导入结果、错误行 | preview 包含成员 |
| E2E-04-008 | 按技能推荐人员 | 点击推荐 | 推荐列表 | 推荐 API/preview 有技能依据 |
| E2E-04-009 | 从项目成员一键选择 | 点击项目成员 | 成员表格填充 | 成员与项目 API 一致 |
| E2E-04-010 | 成员设置为负责人 | 在成员行设为负责人 | 负责人字段变化 | payload owner_id 更新 |
| E2E-04-011 | 分工表跨部门审批状态 | 添加跨部门成员 | 审批状态列 | preview 返回跨部门审批 |
| E2E-04-012 | 冲突与协调分区 | 进入步骤四 | 直接生效、跨部门、负载冲突、状态冲突分区 | `POST /api/dispatch/preview` 分类一致 |
| E2E-04-013 | 问题项详情 | 展开冲突项 | 影响日期、已有任务、建议动作 | preview 返回同字段 |
| E2E-04-014 | 低风险可直接发布 | 仅低风险任务直接发布 | 发布确认 | `POST /api/dispatch/submit` 成功 |
| E2E-04-015 | 中风险二次确认发布 | 中风险任务发布 | 二次确认 | submit 成功且记录确认 |
| E2E-04-016 | 高风险默认阻断 | 高风险任务直接发布 | 阻断提示 | submit 返回 400/409 |
| E2E-04-017 | 保存草稿 | 派发向导保存草稿 | 草稿结果 | 任务状态 draft |
| E2E-04-018 | 提交协调 | 存在跨部门人员提交协调 | 协调提交结果 | `POST /api/dispatch/submit` 生成 approval |
| E2E-04-019 | 直接发布 | 同部门无冲突任务直接发布 | 发布结果 | 任务状态进行中或待确认 |
| E2E-04-020 | 协调单列表默认视图 | 打开协调中心 | 我发起、待我审批、我部门相关、已完成 | `GET /api/approvals` 返回对应集合 |
| E2E-04-021 | 协调单筛选 | 按类型、状态、部门、人员、任务类型、时间、超时筛选 | 每种筛选 | API 查询匹配 |
| E2E-04-022 | 协调单详情任务信息 | 打开协调单详情 | 顶部摘要、任务信息 | `GET /api/approvals/{id}` 字段一致 |
| E2E-04-023 | 协调单人员占用分析 | 查看占用分析 | 当前占用、新任务、超载、重点任务、休假 | 详情 API 含分析 |
| E2E-04-024 | 协调同意 | 审批人同意 | 审批意见、状态变化 | approve API 成功 |
| E2E-04-025 | 协调拒绝 | 审批人拒绝并填原因 | 原因、状态变化 | reject API 成功；发起人待办 |
| E2E-04-026 | 调整后同意 | 修改日期/投入后同意 | 调整表单、结果 | adjust API 成功；任务分工更新 |
| E2E-04-027 | 推荐他人 | 审批时推荐替代人员 | 推荐表单、结果 | 调整建议入记录 |
| E2E-04-028 | 升级协调 | 点击升级协调 | 升级确认、状态 | escalate API 成功 |
| E2E-04-029 | 会议研讨记录 | 新增会议记录 | 记录表单、列表 | meeting-records API 成功 |
| E2E-04-030 | 后补填报确认通过 | 发起人打开后补确认页确认 | 后补内容、通过结果 | 任务生效 |
| E2E-04-031 | 后补填报修改后确认 | 修改周期/成员后确认 | 修改前后 | 任务字段更新 |
| E2E-04-032 | 后补填报退回补充 | 填退回原因 | 退回提示 | 待办生成给负责人 |
| E2E-04-033 | 后补填报取消任务 | 点击取消任务 | 确认、取消后状态 | 任务状态 canceled |
| E2E-04-034 | 派发记录页签 | 在任务详情查看派发记录 | 派发历史 | 审批/dispatch 记录一致 |

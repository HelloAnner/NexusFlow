# 10 智能工具台

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-10-001 | 工具台首页加载 | 打开工具台 | 搜索、分类、常用、分类区、推荐 | `GET /api/tools` |
| E2E-10-002 | 工具搜索 | 输入工具名 | 搜索结果 | tools 查询匹配 |
| E2E-10-003 | 工具分类筛选 | 按常用、智能体、外部、内嵌等筛选 | 结果 | category 匹配 |
| E2E-10-004 | 我的常用 | 收藏工具后查看常用 | 收藏状态 | 工具偏好可读回 |
| E2E-10-005 | 推荐工具 | 查看右侧推荐 | 推荐列表 | 推荐规则或 tools 字段一致 |
| E2E-10-006 | 工具卡打开 | 点击打开 | 外链或内嵌打开状态 | `POST /api/tools/{id}/usage` 记录 |
| E2E-10-007 | 工具收藏 | 点击收藏/取消 | 状态变化 | 用户偏好记录 |
| E2E-10-008 | 工具详情页 | 打开工具详情 | 名称、描述、场景、可见范围、任务类型、打开按钮、使用记录 | `GET /api/tools/{id}` |
| E2E-10-009 | 从任务详情进入工具 | 在任务详情点击推荐工具 | 当前任务上下文 | `POST /api/tools/{id}/context` 返回上下文 |
| E2E-10-010 | 可传递资料展示 | 任务资料页进入工具 | 可传递资料列表 | context 包含资料 |
| E2E-10-011 | 工具使用记录生成 | 打开工具后看记录 | 使用时间、使用人、来源、任务 | `GET /api/tools/{id}/usage` |
| E2E-10-012 | 科研任务工具推荐 | 科研任务详情 | 论文辅助、专利辅助、资料整理 | 推荐规则匹配 |
| E2E-10-013 | 文职任务工具推荐 | 报告/PPT 任务详情 | PPT、报告撰写、格式转换 | 推荐规则匹配 |
| E2E-10-014 | 出差任务工具推荐 | 出差任务详情 | 会议纪要、报告模板 | 推荐规则匹配 |
| E2E-10-015 | 技术支持任务工具推荐 | 技术支持任务详情 | 问答助手、需求整理 | 推荐规则匹配 |
| E2E-10-016 | 工具配置页加载 | 管理员打开工具配置 | 工具列表、编辑区 | tools API |
| E2E-10-017 | 新建外链工具 | 填名称、分类、描述、入口类型外链、地址、可见角色组织、任务类型、推荐、启用 | 表单、列表 | `POST /api/tools` 成功 |
| E2E-10-018 | 外链工具入口必填 | 外链工具不填地址 | 错误提示 | API 返回 400 |
| E2E-10-019 | 新建内嵌工具安全策略 | 内嵌工具配置安全策略 | 表单、结果 | tools payload 含策略 |
| E2E-10-020 | 内嵌工具缺安全策略阻断 | 不配置安全策略提交 | 错误提示 | API 返回 400 |
| E2E-10-021 | 编辑工具 | 修改描述、范围、启用状态 | 修改前后 | `PATCH /api/tools/{id}` |
| E2E-10-022 | 停用工具从首页消失 | 停用工具后返回工具台 | 不再展示 | tools 列表不返回或 status disabled |
| E2E-10-023 | 停用工具从任务推荐消失 | 停用推荐工具后打开任务详情 | 推荐消失 | context/recommendation 不含 |
| E2E-10-024 | 使用记录筛选 | 按工具、使用人、组织、来源、任务、时间筛选 | 结果 | usage API 匹配 |
| E2E-10-025 | 权限不足不可见工具 | 普通用户查看受限工具 | 工具隐藏 | API 不返回或 403 |

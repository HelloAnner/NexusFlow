# 07 首页与看板

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-07-001 | 员工首页布局 | 员工登录首页 | 欢迎、待办、今日任务、风险、快捷入口 | `GET /api/dashboard` 返回员工数据 |
| E2E-07-002 | 任务负责人首页 | 负责人登录首页 | 负责任务、成员完成率、风险数量 | dashboard 与 tasks 聚合一致 |
| E2E-07-003 | 部门主任首页 | 部门主任登录首页 | 本部门任务、部门人员风险 | 数据范围为部门 |
| E2E-07-004 | 中心领导首页 | 中心领导登录首页 | 中心重点任务和风险 | 数据范围为中心 |
| E2E-07-005 | SA 首页布局 | SA 登录首页或后台首页 | 系统状态、待审核、邀请、配置、权限风险 | `GET /api/admin/dashboard`、`GET /api/dashboard` |
| E2E-07-006 | 待审核用户首页 | 待审核用户登录 | 审核状态页、无业务组件 | 业务 API 返回 403 |
| E2E-07-007 | 我的待办组件 | 首页查看待办 | 待办类型、标题、来源、截止、风险、处理按钮 | `GET /api/todos` 与组件一致 |
| E2E-07-008 | 待办按类型筛选 | 筛审批、进度、验收、资料、注册、配置、权限风险 | 每种筛选 | todos type 匹配 |
| E2E-07-009 | 隐藏已完成待办 | 切换隐藏已完成 | 切换前后 | todos status=open |
| E2E-07-010 | 待办标题跳详情 | 点击待办标题 | 目标页面 | action_url 或对象详情可访问 |
| E2E-07-011 | 待办处理按钮跳具体位置 | 点击处理 | 审批/分工/验收/资料锚点 | 目标对象 API 成功 |
| E2E-07-012 | 我的任务员工视角 | 员工首页任务组件 | 本人分工、今日投入、进度 | assignments 数据一致 |
| E2E-07-013 | 我的任务负责人视角 | 负责人首页任务组件 | 负责任务、成员完成率 | tasks owner 匹配 |
| E2E-07-014 | 我的任务管理者视角 | 管理者首页任务组件 | 范围内重点任务 | 数据范围裁剪 |
| E2E-07-015 | 风险提醒组件 | 查看延期、冲突、审批超时、成果未提交、资料缺失 | 风险列表 | conflicts/todos/resources 状态一致 |
| E2E-07-016 | 风险推荐动作跳转 | 点击推荐动作 | 目标处理页 | 目标 API 可用 |
| E2E-07-017 | 甘特摘要员工视角 | 首页甘特摘要 | 个人时间线 | `GET /api/gantt/summary` |
| E2E-07-018 | 甘特摘要负责人视角 | 切换负责人账号 | 负责任务和分工 | gantt 数据范围 |
| E2E-07-019 | 甘特摘要部门视角 | 部门主任账号 | 部门人员行视图 | gantt 数据范围 |
| E2E-07-020 | 甘特摘要时间切换 | 切本周、本月 | 两种摘要 | API 时间范围匹配 |
| E2E-07-021 | 甘特摘要展开 | 点击展开 | 完整甘特图页 | `GET /api/gantt` 成功 |
| E2E-07-022 | 快捷入口权限展示 | 不同角色查看快捷入口 | 入口差异 | `GET /api/dashboard/role-entry` 一致 |
| E2E-07-023 | 新建任务快捷入口 | 点击新建任务 | 新建任务页 | users/projects options 成功 |
| E2E-07-024 | 新建项目快捷入口 | 点击新建项目 | 新建项目页或弹窗 | projects POST 权限检查 |
| E2E-07-025 | 生成邀请链接快捷入口 | SA 点击生成邀请链接 | 邀请链接页 | invitations API 成功 |
| E2E-07-026 | 处理待审核注册快捷入口 | SA 点击待审核 | 待审核页 | registrations API 成功 |
| E2E-07-027 | 新建休假快捷入口 | 员工点击新建休假 | 休假任务表单 | task type=leave |
| E2E-07-028 | 填报进度快捷入口 | 员工点击填报进度 | 分工进度页 | progress API 可用 |
| E2E-07-029 | 上传资料快捷入口 | 点击上传资料 | 上传弹窗 | resource upload API |
| E2E-07-030 | 打开工具台快捷入口 | 点击工具台 | 工具台 | tools API |
| E2E-07-031 | SA 系统状态组件 | 点击系统状态 | 运行状态页 | `GET /api/config/runtime-status` |
| E2E-07-032 | SA 邀请异常组件 | 点击邀请异常 | 邀请链接列表 | links API |
| E2E-07-033 | SA 配置异常组件 | 点击配置异常 | 配置中心 | config modules |
| E2E-07-034 | SA 权限审计风险组件 | 点击权限风险 | 审计日志 | audit API |
| E2E-07-035 | 最近动态权限裁剪 | 不同角色查看动态 | 动态列表差异 | `GET /api/dashboard/recent-activities` 裁剪 |
| E2E-07-036 | 移动端首页优先级 | 移动视口打开首页 | 待办、任务、风险、入口、甘特、动态顺序 | API 与桌面一致 |

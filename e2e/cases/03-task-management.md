# 03 任务管理

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-03-001 | 不同角色默认任务视图 | 员工、负责人、部门主任、中心领导、管理员分别登录任务列表 | 每个角色默认列表 | `GET /api/tasks` 范围和排序符合角色 |
| E2E-03-002 | 任务状态筛选 | 逐项筛选草稿、待协调、待确认、进行中、待验收、已完成、已归档 | 每种筛选结果 | API `status` 匹配 |
| E2E-03-003 | 任务类型筛选 | 筛科研、报告、出差、休假等 | 筛选结果 | 任务 `type/sub_type` 匹配 |
| E2E-03-004 | 任务优先级筛选 | 筛低、中、高、紧急 | 筛选结果 | `priority` 匹配 |
| E2E-03-005 | 负责人和成员筛选 | 分别按负责人、成员筛选 | 筛选结果 | owner/member 匹配 |
| E2E-03-006 | 所属组织和项目筛选 | 按组织、项目筛选 | 筛选结果 | org/project 匹配 |
| E2E-03-007 | 时间范围筛选 | 选择本周、本月、自定义 | 筛选结果 | 起止时间与查询一致 |
| E2E-03-008 | 风险和待办筛选 | 只看延期、冲突、资料缺失、有待办 | 筛选结果 | 风险字段与待办 API 一致 |
| E2E-03-009 | 隐藏项目任务未授权不可见 | 未授权用户搜索隐藏任务 | 空态 | API 不返回隐藏任务 |
| E2E-03-010 | 批量导出 | 勾选多任务导出 | 选择、下载提示 | 导出 API 返回文件或任务 |
| E2E-03-011 | 已完成任务批量归档 | 勾选已完成任务归档 | 确认、归档后列表 | `POST /api/tasks/{id}/archive` 成功 |
| E2E-03-012 | 未完成任务批量归档被阻断 | 勾选进行中任务归档 | 阻断提示 | API 返回 400/409 |
| E2E-03-013 | 批量关注和取消关注 | 勾选任务关注、取消 | 状态变化 | 关注状态可读回 |
| E2E-03-014 | 新建任务基本信息 | 打开新建向导，填写名称、类型、级别、优先级、组织、项目、可见性、日期、概述、成果要求 | 第一步表单 | `POST /api/tasks` 或草稿保存成功 |
| E2E-03-015 | 类型切换加载模板 | 切换任务类型 | 模板字段变化 | `GET /api/config/task_template` 或任务模板数据匹配 |
| E2E-03-016 | 休假类型隐藏无关字段 | 选择休假 | 项目/成果字段隐藏 | 提交 payload 不含无关字段 |
| E2E-03-017 | 隐藏可见性必须配置授权 | 选择隐藏且不配置授权提交 | 错误提示 | API 返回 400 |
| E2E-03-018 | 截止早于开始禁止保存 | 输入非法日期 | 字段错误 | API 返回 400 |
| E2E-03-019 | 人员选择按姓名搜索 | 第二步搜索成员 | 搜索结果 | `GET /api/users?q=...` 匹配 |
| E2E-03-020 | 人员选择按组织筛选 | 选择组织 | 成员候选 | 候选人属于组织 |
| E2E-03-021 | 人员选择按技能筛选 | 选择技能 | 成员候选 | 候选人有技能 |
| E2E-03-022 | 从项目成员选择 | 选择关联项目后一键选择项目成员 | 项目成员候选 | 项目成员 API 匹配 |
| E2E-03-023 | 系统推荐候选人 | 打开推荐 | 推荐列表 | 推荐依据含负载/技能 |
| E2E-03-024 | 单成员自动生成同名分工 | 选择单成员进入分工步骤 | 自动分工行 | 分工 payload 包含该成员 |
| E2E-03-025 | 多成员分工草案 | 为多成员填写工作内容、类型、周期、投入、成果 | 分工表格 | `POST /api/tasks/{id}/assignments` 成功 |
| E2E-03-026 | 跨部门成员需要审批 | 选择外部门成员发布 | 协调提示 | `POST /api/dispatch/preview` 返回需审批 |
| E2E-03-027 | 冲突检查展示工时超载 | 制造超载分工并检查 | 冲突检查页 | `POST /api/workload/preview` 返回 overload |
| E2E-03-028 | 冲突检查展示全天重叠 | 选择全天占用人员 | 冲突检查页 | preview 返回 all_day |
| E2E-03-029 | 冲突检查展示休假/出差 | 选择休假人员 | 冲突检查页 | preview 返回 unavailable |
| E2E-03-030 | 冲突检查展示隐藏授权问题 | 隐藏任务未授权成员 | 冲突检查页 | preview 返回 visibility issue |
| E2E-03-031 | 冲突建议动作 | 查看调整人员、时间、投入、发起协调、保存草稿 | 每个建议入口 | API action 可执行或阻断原因明确 |
| E2E-03-032 | 保存草稿 | 新建任务保存草稿 | 草稿详情 | `GET /api/tasks/{id}` 状态 draft |
| E2E-03-033 | 草稿提交协调 | 草稿详情点击提交 | 状态变化 | `POST /api/tasks/{id}/submit` 成功 |
| E2E-03-034 | 待协调撤回 | 待协调任务点击撤回 | 确认、状态变化 | 状态回到草稿或取消协调 |
| E2E-03-035 | 待确认确认发布 | 负责人确认 | 任务进入进行中 | `POST /api/tasks/{id}/confirm` 成功 |
| E2E-03-036 | 进行中更新进度 | 任务详情更新进度 | 进度条变化 | `POST /api/assignments/{id}/progress` 成功 |
| E2E-03-037 | 进行中暂停 | 点击暂停 | 状态标签变化 | `POST /api/tasks/{id}/pause` 成功 |
| E2E-03-038 | 进行中发起变更 | 修改周期/成员 | 变更审批页 | 审批单生成 |
| E2E-03-039 | 进行中提交验收 | 提交成果后验收 | 待验收状态 | `POST /api/tasks/{id}/submit-acceptance` 成功 |
| E2E-03-040 | 待验收通过 | 验收人点击通过 | 完成状态 | `POST /api/tasks/{id}/accept` 成功 |
| E2E-03-041 | 待验收退回必须填原因 | 不填原因退回 | 错误提示 | API 返回 400 |
| E2E-03-042 | 待验收退回生成待办 | 填原因退回 | 状态、待办 | `POST /api/tasks/{id}/reject` 成功；`GET /api/todos` 有待办 |
| E2E-03-043 | 已完成归档 | 点击归档 | 已归档状态 | `POST /api/tasks/{id}/archive` 成功 |
| E2E-03-044 | 已归档只读 | 打开已归档任务 | 操作禁用 | 写 API 返回 403/409 |
| E2E-03-045 | 概览页签展示完整 | 打开详情概览 | 概述、成果、里程碑、成员、风险、动态 | 详情 API 字段齐全 |
| E2E-03-046 | 分工页签新增分工 | 详情中新增分工 | 新增表单、结果 | 分工可读回 |
| E2E-03-047 | 分工页签编辑分工 | 修改工作内容、投入 | 前后对比 | PATCH assignment 成功 |
| E2E-03-048 | 分工提交成果 | 成员提交成果 | 成果展示 | submit-result 成功 |
| E2E-03-049 | 分工确认成果 | 负责人确认 | 分工状态完成 | assignment 状态完成 |
| E2E-03-050 | 分工退回修改 | 负责人退回 | 退回原因和待办 | 分工状态退回 |
| E2E-03-051 | 任务甘特页签 | 打开任务甘特 | 任务、里程碑、分工三层时间线 | 详情 assignments 时间一致 |
| E2E-03-052 | 资料页签必需资料 | 打开资料页签 | 必需资料缺失/完整状态 | `GET /api/resources/check-requirements` 一致 |
| E2E-03-053 | 审批页签 | 打开审批页签 | 协调、强制、变更、隐藏授权 | `GET /api/approvals` 匹配 |
| E2E-03-054 | 日志页签 | 打开日志页签 | 状态、字段、评论、审批、资料、验收记录 | 事件/审计可查 |
| E2E-03-055 | 验收页要求补充资料 | 必需资料缺失时验收 | 缺失提示、补充入口 | 验收 API 阻断 |
| E2E-03-056 | 验收转交 | 选择其他验收人 | 转交表单、结果 | 验收人更新 |

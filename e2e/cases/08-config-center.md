# 08 配置中心

## 用例

| ID | 场景 | 浏览器步骤 | 必须截图 | API 后验 |
| --- | --- | --- | --- | --- |
| E2E-08-001 | 配置中心首页加载 | 进入配置中心 | 模块卡片、健康状态 | `GET /api/config/modules` 返回模块 |
| E2E-08-002 | 配置异常展示 | 制造任务类型无模板、审批无审批人等 | 异常卡片 | config/modules 标记异常 |
| E2E-08-003 | 任务类型树加载 | 打开任务类型配置 | 左树右详情 | `GET /api/config/task_type` |
| E2E-08-004 | 新增任务类型 | 新增类型，配置负载/状态/模板/审批/资料 | 表单、结果 | config draft/publish 后读回 |
| E2E-08-005 | 编辑任务类型 | 修改类型字段 | 修改前后 | 版本记录新增 |
| E2E-08-006 | 停用任务类型 | 停用类型 | 停用确认 | 历史任务仍展示该类型 |
| E2E-08-007 | 复制任务类型 | 复制已有类型 | 复制结果 | 新配置与源一致 |
| E2E-08-008 | 任务模板字段组件库 | 打开模板编辑器 | 左侧组件、中间画布、右侧属性 | config 返回模板结构 |
| E2E-08-009 | 模板添加文本字段 | 拖入单行/多行文本 | 画布和属性 | 草稿 JSON 包含字段 |
| E2E-08-010 | 模板添加日期字段 | 拖入日期/日期范围 | 画布和属性 | 字段类型正确 |
| E2E-08-011 | 模板添加人员组织项目字段 | 拖入人员、组织、项目 | 画布和属性 | 字段类型正确 |
| E2E-08-012 | 模板添加枚举数字文件字段 | 拖入枚举、数字、文件 | 画布和属性 | 字段类型正确 |
| E2E-08-013 | 模板添加分工/里程碑表格 | 拖入表格 | 画布和属性 | 表格 schema 正确 |
| E2E-08-014 | 字段属性校验 | 设置必填、默认值、说明、可见角色、可编辑角色、校验规则 | 属性面板 | 草稿 JSON 正确 |
| E2E-08-015 | 审批规则列表 | 打开审批规则 | 规则列表 | `GET /api/config/approval_rule` |
| E2E-08-016 | 新增审批规则 | 配置触发条件、层级、审批人来源、超时、拒绝动作、转交 | 表单、结果 | 发布后规则可读回 |
| E2E-08-017 | 审批规则无审批人异常 | 保存无审批人规则 | 异常提示 | API 返回 400 或模块异常 |
| E2E-08-018 | 告警规则配置 | 新增告警名称、类型、条件、提前时间、等级、对象、待办、启用 | 表单、结果 | config 读回 |
| E2E-08-019 | 首页视图配置 | 按角色配置组件启用、排序、范围、时间、入口 | 配置页、预览 | `GET /api/dashboard/role-entry` 生效 |
| E2E-08-020 | 首页角色预览 | 预览 SA、中心、部门、负责人、员工、待审核 | 每个预览 | dashboard API 数据范围匹配 |
| E2E-08-021 | 项目类型配置 | 新增/停用项目类型 | 配置和历史项目 | 历史项目仍展示 |
| E2E-08-022 | 项目成员身份配置 | 新增/停用身份 | 配置和历史成员 | 历史身份保留 |
| E2E-08-023 | 隐藏项目授权范围配置审计 | 修改授权范围 | 二次确认、审计 | audit 有记录 |
| E2E-08-024 | 项目资料要求配置 | 配置项目必需资料 | 配置页 | resources check 使用新规则 |
| E2E-08-025 | 项目归档校验规则 | 配置归档前校验 | 配置页 | archive API 按规则阻断/通过 |
| E2E-08-026 | 系统运行状态只读 | 打开运行状态 | 环境、端口、DB、Redis、S3、搜索、上传、版本、启动时间 | `GET /api/config/runtime-status` |
| E2E-08-027 | 运行状态不泄露密钥 | 查看页面和 API | 无密码/token/secret | 响应不含敏感字段 |
| E2E-08-028 | 运行异常展示摘要 | 模拟组件异常 | 错误摘要、检查时间 | API status degraded |
| E2E-08-029 | 不展示 Docker 本机配置 | 打开运行状态 | 无 Docker 启动引导 | 页面符合服务器部署约定 |
| E2E-08-030 | 邀请注册默认有效期配置 | 修改有效期 | 配置页、生成链接 | 新链接 expires_at 符合 |
| E2E-08-031 | 邀请最大使用次数配置 | 修改最大次数 | 配置页、链接 | max_uses 符合 |
| E2E-08-032 | 注册字段配置 | 配置必填/选填字段 | 注册页变化 | register API 校验同步 |
| E2E-08-033 | 待审核策略配置 | 修改是否审核、审核人来源 | 注册后结果 | registration 状态符合策略 |
| E2E-08-034 | 修改审核策略写审计 | 发布策略 | 审计日志 | audit 记录 |
| E2E-08-035 | 角色入口配置 | 配置默认首页、菜单、组件、数据范围、时间、入口、待办、切换视角 | 配置和预览 | role-entry API 生效 |
| E2E-08-036 | 配置草稿保存 | 保存但不发布 | 草稿提示 | `POST /api/config/{namespace}/draft` 返回 id |
| E2E-08-037 | 配置发布 | 发布草稿并填原因 | 发布结果 | `POST /api/config/{namespace}/publish`；版本新增 |
| E2E-08-038 | 配置停用 | 停用配置版本 | 停用确认 | `POST /api/config/{namespace}/disable` |
| E2E-08-039 | 配置版本记录 | 打开版本记录 | 创建、发布、发布人、原因 | `GET /api/config/versions` |
| E2E-08-040 | 权限不足不能配置 | 普通用户打开配置中心 | 菜单隐藏或 403 | config 写 API 返回 403 |

# 配置中心页面设计

## 1. 页面清单

- 配置中心首页。
- 任务类型配置页。
- 任务模板编辑页。
- 审批规则配置页。
- 告警规则配置页。
- 资料类型配置页。
- 首页视图配置页。
- 项目配置页。
- 邀请注册配置页。
- 角色入口配置页。
- 系统运行状态页。
- 配置版本记录页。

## 2. 配置中心首页

### 2.1 页面目标

集中展示所有配置模块和配置健康状态。

### 2.2 模块卡片

每个配置模块展示：

- 模块名称。
- 配置数量。
- 最近修改时间。
- 最近修改人。
- 是否存在异常。

### 2.3 配置异常

异常示例：

- 任务类型未绑定模板。
- 项目类型未配置成员身份。
- 审批规则无审批人。
- 必需资料类型已停用。
- 首页组件引用了停用工具。

## 3. 任务类型配置页

布局：

- 左侧类型树。
- 右侧类型详情。

详情字段：

- 类型名称。
- 父类型。
- 是否启用。
- 是否参与负载。
- 是否改变人员状态。
- 默认模板。
- 默认审批规则。
- 默认资料要求。

操作：

- 新增类型。
- 编辑类型。
- 停用类型。
- 复制类型。

## 4. 任务模板编辑页

### 4.1 页面结构

- 左侧：字段组件库。
- 中间：表单画布。
- 右侧：字段属性。

### 4.2 字段组件

- 单行文本。
- 多行文本。
- 日期。
- 日期范围。
- 人员。
- 组织。
- 项目。
- 枚举。
- 数字。
- 文件。
- 分工表格。
- 里程碑表格。

### 4.3 字段属性

- 字段名。
- 字段编码。
- 是否必填。
- 默认值。
- 帮助说明。
- 可见角色。
- 可编辑角色。
- 校验规则。

## 5. 审批规则配置页

### 5.1 规则列表字段

- 规则名称。
- 触发场景。
- 适用任务类型。
- 审批人来源。
- 是否启用。
- 最近修改时间。

### 5.2 规则编辑

配置项：

- 触发条件。
- 审批层级。
- 审批人来源。
- 超时策略。
- 拒绝后动作。
- 是否允许转交。

## 6. 告警规则配置页

配置项：

- 告警名称。
- 告警类型。
- 触发条件。
- 提前提醒时间。
- 风险等级。
- 提醒对象。
- 是否生成待办。
- 是否启用。

## 7. 首页视图配置页

按角色配置首页：

- 组件启用。
- 组件排序。
- 默认数据范围。
- 默认时间范围。
- 快捷入口。

支持预览不同角色首页。

## 8. 项目配置页

配置内容：

- 项目类型。
- 项目级别。
- 项目成员身份。
- 项目状态。
- 隐藏项目授权范围。
- 项目资料要求。
- 项目归档校验规则。

页面规则：

- 停用项目类型后，历史项目仍正常展示。
- 项目成员身份停用后，历史成员身份保留。
- 隐藏项目授权范围变更需要写入权限审计。

## 9. 系统运行状态页

系统运行状态页只读展示 `.env` 相关基础组件状态，不提供密钥编辑能力。

展示内容：

- 当前运行环境。
- 监听端口，默认 `8089`。
- 数据库连接状态。
- Redis 连接状态。
- 对象存储连接状态。
- 搜索后端。
- 上传大小限制。
- 构建版本。
- 启动时间。

规则：

- 不展示连接密码、密钥和 token。
- 连接异常时展示错误摘要和最近检查时间。
- 不提供 Docker、容器或本机组件启动配置。

## 10. 邀请注册配置页

配置内容：

- 默认有效期。
- 最大使用次数。
- 是否允许批量邀请。
- 注册字段。
- 待审核策略。
- 审核人来源。
- 注册成功默认入口。

页面规则：

- 密钥和 token 不在配置页展示。
- 模板停用后已生成链接同步失效或提示处理。
- 修改审核策略需要写入审计。

## 11. 角色入口配置页

页面目标：

配置不同角色登录后的导航、首页组件、默认数据范围和快捷入口。

配置内容：

- 角色。
- 默认首页。
- 菜单集合。
- 首页组件。
- 默认时间范围。
- 默认数据范围。
- 快捷入口。
- 待办类型。
- 是否可切换角色视角。

内置角色：

- SA 超级管理员。
- 中心主任/书记。
- 中心副主任/技术总监。
- 部门主任/技术监督。
- 部门副主任。
- 项目负责人。
- 任务负责人。
- 员工。
- 待审核用户。

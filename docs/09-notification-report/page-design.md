# 通知、待办与报表页面设计

## 1. 页面清单

- 待办中心。
- 通知中心。
- 报表首页。
- 任务总览报表。
- 人员负载报表。
- 资料归档报表。
- 协调统计报表。

## 2. 待办中心

### 2.1 筛选项

- 待办类型。
- 状态。
- 风险等级。
- 关联任务。
- 发起人。
- 截止时间。

### 2.2 列表字段

- 待办标题。
- 类型。
- 关联对象。
- 发起人。
- 截止时间。
- 风险等级。
- 状态。
- 操作按钮。

### 2.3 操作

- 立即处理。
- 标记已读。
- 转交，按规则控制。
- 查看来源。

处理按钮必须跳转到具体页面：

- 审批待办跳协调单。
- 进度待办跳分工填报。
- 验收待办跳任务验收。
- 资料待办跳任务资料页签。

## 3. 通知中心

通知分类：

- 任务。
- 审批。
- 风险。
- 资料。
- 系统。

列表展示：

- 通知标题。
- 内容摘要。
- 来源对象。
- 时间。
- 是否已读。

## 4. 报表首页

卡片：

- 进行中任务数。
- 高风险任务数。
- 本周到期任务数。
- 待审批协调数。
- 平均人员负载。
- 资料完整率。

图表：

- 任务状态分布。
- 任务类型分布。
- 部门负载排行。
- 重点任务进度。

## 5. 任务总览报表

筛选：

- 组织。
- 项目。
- 任务类型。
- 时间范围。
- 负责人。
- 状态。

图表：

- 状态堆叠图。
- 月度新增和完成趋势。
- 延期任务列表。

## 6. 人员负载报表

展示：

- 人员。
- 部门。
- 周期标准工时。
- 预计投入工时。
- 负载率。
- 超载天数。
- 全天占用天数。

支持从报表跳转人员详情和负载日历。

## 7. 资料归档报表

展示：

- 任务。
- 任务类型。
- 必需资料数。
- 已确认资料数。
- 缺失资料。
- 完整率。

支持跳转任务资料页签补充资料。

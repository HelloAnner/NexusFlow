# 项目详情（Project Detail）

> 参考 Linear Project View 和 Figma 右侧属性面板，项目详情以右侧面板形式呈现，支持快速预览和进入项目空间。

## 1. 页面目标

在项目中心点击项目后，从右侧滑出上下文面板，展示项目概览、关键指标、成员、最近任务、风险等。

## 3. Overview Tab

- 项目概述（块编辑器）。
- 关键指标：任务总数、进行中、有风险、已完成、成员数。
- 最近动态（前 5 条）。
- 快捷操作：进入项目空间 / 编辑 / 归档。

## 4. Members Tab

- 成员头像列表。
- 点击成员进入人员详情面板。
- 提供添加成员入口。

## 5. Tasks Tab

- 项目内最近 10 个任务。
- 点击任务打开任务详情面板。
- “查看全部”进入项目空间 Tasks Tab。

## 6. Files Tab

- 项目级资料列表。
- 支持上传新资料。

## 7. Risks Tab

- 项目内风险列表。
- 推荐处理动作。

## 8. Activity Tab

- 项目创建、成员变更、状态变更、授权变更记录。

## 9. 操作区

- **进入项目空间**：主按钮。
- **编辑**：进入编辑模式。
- **归档 / 取消归档**：根据状态。
- **删除**：需二次确认。

## 10. 面板交互细节

- 面板打开后不改变主列表筛选和滚动位置，关闭后用户回到原项目行。
- 面板宽度可拖拽，默认 560px，最小 420px，最大不超过视口 70%；用户固定后跨页面保留。
- “进入项目空间”是主操作，始终放在顶部右侧；当项目被归档时按钮文案变为“查看项目空间”。
- 编辑模式不进入单独页面，字段就地编辑；用户修改多个字段后可逐项保存，也可统一保存。
- Overview 中的项目概述为空时，展示轻量占位“添加项目目标、范围和交付物”，只对可编辑用户显示。
- 最近动态只展示前 5 条，点击“查看全部”切换到 Activity Tab。

## 11. 数据与跳转

- Members Tab 点击成员时打开人员详情面板，如果当前项目面板已固定，则允许形成二级面板或替换当前面板，交互需统一。
- Tasks Tab 点击任务打开 Task Detail 面板；返回时仍停留在 Project Detail 的 Tasks Tab。
- Files Tab 上传文件默认关联当前项目，并允许继续选择关联任务。
- Risks Tab 的推荐处理动作如果涉及调整任务，点击后打开任务详情或新建协调流程，不在风险列表内完成复杂编辑。
- Activity Tab 中每条变更都要包含操作人、时间、变更对象和变更前后摘要。

## 12. 边界状态

- 项目无成员时，Members Tab 展示“添加成员”入口；无权限用户只看到空态说明。
- 项目无任务时，Tasks Tab 展示“新建任务”和“从模板创建任务”。
- 用户无权查看项目文件时，Files Tab 隐藏列表并说明可见范围，不展示空白。
- 项目被归档后，编辑入口默认禁用，只保留取消归档和查看历史。
- 项目加载失败时面板保留关闭按钮和重试按钮，不能让遮罩卡死页面。

## 建议组件

项目详情以右侧面板形式呈现，信息层级和编辑顺滑度是关键。推荐面板容器使用 shadcn/ui `Sheet`（桌面端右侧滑出，宽度 560px；移动端使用 **vaul** 全屏抽屉）。所有 shadcn 组件通过 CSS 变量映射到 NexusFlow 设计系统：`Sheet` 背景 `--bg-secondary`（#FFFFFF），遮罩 `bg-black/20`，内容区文字 `--text-primary`（#1A1A1A）/`--text-muted`（#7A7A7A），分割线 `--border-subtle`（rgba(0,0,0,0.05)）。避免使用 shadcn 默认的彩色焦点环，统一改为 `--text-primary` 的 `outline` 样式。

面板顶部操作区使用 shadcn/ui `Button`：主按钮“进入项目空间”填充 `--text-primary`；“编辑”使用 `outline` 变体；“更多”使用 `ghost` 变体 + `DropdownMenu`。归档/删除等危险操作放入 `DropdownMenu` 并用 `--error` 文字色提示，删除使用 `AlertDialog` 二次确认。关闭按钮使用 `ghost` 变体，图标 `--text-muted`。

Tab 导航使用 shadcn/ui `Tabs`，包含 Overview、Members、Tasks、Files、Risks、Activity。Tab 激活态下划线 `--text-primary`，内容区背景与面板背景一致，避免多层嵌套造成视觉噪音。

**Overview Tab** 的项目概述建议自研轻量块编辑器，优先基于 **BlockNote** 或 **Editor.js**，支持段落、标题、列表、待办、附件占位。块编辑器工具栏使用 shadcn/ui `Button` 的 `ghost` 变体，图标颜色 `--text-muted`，hover 背景 `--bg-tertiary`。编辑器内容区背景 `--bg-secondary`，占位符颜色 `--text-muted`。如果需求更复杂（如协同光标、 mention），可评估 **Slate** 自研，但成本更高。关键指标卡片建议自研 **Metric Card 组件**，6 宫格或 4 宫格布局，每个卡片背景 `--bg-tertiary`，数字使用 `text-2xl font-semibold text-primary`，标签使用 `text-muted`。不使用彩色背景区分指标，保持中性。最近动态前 5 条使用自研 Activity Feed 组件，时间 `--text-muted`，内容 `--text-primary`，对象链接 hover 下划线。

Members Tab 的成员头像组使用 shadcn/ui `Avatar` 自研 AvatarGroup：头像重叠区域使用 `--bg-secondary` 边框隔离，hover 展开成员名 Tooltip。点击成员进入人员详情面板。提供添加成员入口，使用 shadcn/ui `Dialog` 选择人员。

Tasks Tab 展示最近 10 个任务，使用自研高密度行组件，与 My Work List 视图保持一致，点击任务打开任务详情面板，“查看全部”进入项目空间 Tasks Tab。Files Tab 复用 Files 页面轻量列表，支持上传新资料。Risks Tab 使用 shadcn/ui `Badge`（语义色）+ 自研风险行，推荐处理动作使用 `Button` 的 `ghost` 变体。Activity Tab 使用自研 **Activity Timeline 组件**：左侧竖线 `--border-subtle`，节点使用 `--text-muted` 圆点，时间 `--text-muted`，动态内容 `--text-primary`，对象链接 hover 下划线。

所有加载态使用 shadcn/ui `Skeleton`，骨架颜色通过 CSS 变量映射到 `--bg-tertiary`（#F5F4F2），避免彩色闪烁。面板内所有按钮、链接的交互反馈使用 150ms 过渡，保持全局动画规范。项目概述的块编辑器内容建议以 JSON 格式持久化，便于后续版本对比与导出。

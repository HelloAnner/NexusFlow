# 组织与人员页面设计

## 1. 页面清单

- 组织管理页。
- 人员列表页。
- 人员详情页。
- 人员编辑页。
- 邀请注册人员审核入口。
- 技能标签管理页。
- 人员负载视图入口。

## 2. 组织管理页

### 2.1 页面目标

用于维护组织树、组织负责人和默认审批人，为权限、统计和跨部门协调提供基础。

### 2.2 页面布局

- 左侧：组织树。
- 右侧上方：当前组织基础信息。
- 右侧中部：负责人、副负责人、技术监督、默认审批人。
- 右侧下方：组织人员列表和下级组织列表。

### 2.3 关键操作

- 新增同级组织。
- 新增下级组织。
- 编辑组织。
- 停用组织。
- 查看组织人员。
- 查看组织任务。

### 2.4 交互规则

- 点击组织树节点后，右侧即时切换详情。
- 停用组织前，系统提示影响范围。
- 有人员或下级组织时不允许直接删除。
- 组织负责人为空时，跨部门审批需回退到上级组织默认审批人。

## 3. 人员列表页

### 3.1 页面目标

帮助管理者按组织、技能、项目、状态快速找到人员，并判断其是否适合接收任务。

### 3.2 筛选区

筛选项：

- 组织。
- 工作室。
- 系统角色。
- 人员等级。
- 项目归属。
- 技能标签。
- 工作状态。
- 账号状态。
- 邀请来源。
- 是否待审核。
- 本周负载范围。
- 是否可派发。

### 3.3 列表字段

| 字段 | 说明 |
| --- | --- |
| 姓名 | 点击进入人员详情 |
| 主组织 | 展示到部门 |
| 角色 | 展示主要系统角色 |
| 人员等级 | 管理等级或专业等级 |
| 技能标签 | 最多展示 5 个，超出折叠 |
| 当前任务数 | 进行中任务数量 |
| 本周负载 | 百分比和颜色标记 |
| 工作状态 | 在岗、休假、出差等 |
| 账号状态 | 待审核、启用、禁用、锁定 |
| 操作 | 查看、编辑、发起任务 |

### 3.4 负载颜色

- 0% 到 60%：绿色。
- 60% 到 90%：黄色。
- 90% 到 100%：橙色。
- 超过 100%：红色。
- 全天占用：灰色不可派发标识。

## 4. 人员详情页

### 4.1 顶部信息

展示：

- 姓名、部门、角色、状态。
- 当前本周负载。
- 可派发状态。
- 快捷操作：编辑、发起任务、查看甘特图。

### 4.2 页签

#### 基础信息

- 主组织。
- 工作室。
- 人员等级。
- 系统角色。
- 每日标准工时。
- 联系方式。

#### 项目归属

- 项目名称。
- 项目身份。
- 负责工作。
- 加入时间。
- 当前状态。

#### 技能标签

- 按分组展示技能。
- 支持管理者编辑。
- 支持展示标签来源和更新时间。

#### 当前任务

- 当前参与任务。
- 当前负责分工。
- 截止时间。
- 风险状态。

#### 负载日历

- 以周或月展示每日占用。
- 点击某天展示任务明细。
- 全天任务显示为整日灰色块。

#### 历史记录

- 组织变更。
- 角色变更。
- 项目归属变更。
- 状态变更。

## 5. 人员编辑页

### 5.1 表单分组

- 基础信息。
- 账号安全。
- 组织归属。
- 角色权限。
- 项目归属。
- 技能标签。
- 工作状态。
- 工时设置。
- 默认首页。
- 审计备注。

### 5.2 校验规则

- 姓名、账号、主组织、角色必填。
- 账号不可重复。
- 离职人员不可设置为任务负责人。
- 每日标准工时必须大于 0。
- 当前用户无权限的角色不可分配。
- 待审核人员通过审核前不可设置为任务负责人。
- 非 SA 用户不可修改账号状态、隐藏项目授权和超出自己范围的角色。

## 6. 邀请注册人员审核入口

人员模块展示待审核注册入口，完整审核流程在 SA 后台或授权管理员后台完成。

列表字段：

- 姓名。
- 登录账号。
- 邀请组织。
- 邀请角色。
- 邀请项目。
- 注册时间。
- 邀请来源。

操作：

- 查看注册资料。
- 审核通过。
- 拒绝。
- 修改组织后通过。
- 修改角色后通过。
- 合并到已有人员。

## 7. 技能标签管理页

### 7.1 页面结构

- 左侧标签分组。
- 右侧标签列表。
- 顶部新增标签按钮。

### 7.2 字段

- 标签名称。
- 标签分组。
- 描述。
- 是否启用。
- 排序。

### 7.3 规则

- 停用标签后，历史人员仍展示该标签，但不可新选择。
- 标签被使用时不允许删除。
- 标签名称在同一分组内不可重复。

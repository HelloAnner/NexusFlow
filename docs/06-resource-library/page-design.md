# 资料库页面设计

## 1. 页面清单

- 资料库列表页。
- 任务资料页签。
- 资料详情页。
- 资料上传弹窗。
- 版本历史弹窗。

## 2. 资料库列表页

### 2.1 页面目标

按权限范围检索和管理任务资料。

### 2.2 筛选项

- 资料名称。
- 资料类型。
- 关联任务。
- 关联项目。
- 上传人。
- 组织。
- 状态。
- 是否阶段成果。
- 是否最终成果。
- 上传时间。

### 2.3 列表字段

| 字段 | 说明 |
| --- | --- |
| 资料名称 | 点击进入详情 |
| 资料类型 | 报告、PPT、数据等 |
| 关联任务 | 可跳转 |
| 上传人 | 当前版本上传人 |
| 版本 | 最新版本 |
| 状态 | 已提交、已确认等 |
| 成果标记 | 阶段成果、最终成果 |
| 可见范围 | 简要展示 |
| 更新时间 | 最新版本时间 |

### 2.4 操作

- 上传资料。
- 下载。
- 查看详情。
- 上传新版本。
- 标记成果。
- 确认资料。
- 退回资料。

## 3. 任务资料页签

### 3.1 页面结构

- 必需资料清单。
- 已上传资料列表。
- 阶段成果区。
- 最终成果区。

### 3.2 必需资料清单

展示：

- 资料类型。
- 是否必填。
- 当前状态。
- 缺失提示。
- 上传入口。

### 3.3 资料卡片

卡片展示：

- 资料名称。
- 类型。
- 上传人。
- 版本。
- 状态。
- 成果标记。
- 更新时间。

操作：

- 预览。
- 下载。
- 新版本。
- 设为阶段成果。
- 设为最终成果。

## 4. 资料详情页

### 4.1 顶部信息

- 资料名称。
- 类型。
- 状态。
- 最新版本。
- 上传人。
- 关联任务。
- 关联分工。

### 4.2 内容区

页签：

- 预览。
- 版本历史。
- 关联任务。
- 权限范围。
- 操作日志。

## 5. 资料上传弹窗

字段：

- 资料类型。
- 文件。
- 关联任务。
- 关联分工。
- 版本说明。
- 是否阶段成果。
- 是否最终成果。

校验：

- 资料类型必填。
- 文件必填。
- 上传人必须有任务访问权限。
- 最终成果标记需要负责人或验收人权限。

# 任务管理页面设计

## 1. 页面清单

- 任务列表页。
- 新建/编辑任务页。
- 任务详情页。
- 分工管理页签。
- 任务甘特图页签。
- 任务验收页。
- 任务变更记录页签。

## 2. 任务列表页

### 2.1 页面目标

让不同角色快速找到需要处理的任务，并优先暴露风险、延期和待办。

### 2.2 默认视图

| 角色 | 默认范围 | 默认排序 |
| --- | --- | --- |
| 员工 | 我参与的任务 | 截止时间升序 |
| 任务负责人 | 我负责的任务 | 风险优先 |
| 部门主任 | 本部门任务 | 风险和截止时间优先 |
| 中心领导 | 中心重点任务 | 风险等级优先 |
| 管理员 | 全部任务 | 创建时间倒序 |

### 2.3 筛选项

- 任务状态。
- 任务类型。
- 任务级别。
- 优先级。
- 负责人。
- 成员。
- 所属组织。
- 关联项目。
- 时间范围。
- 风险状态。
- 是否隐藏项目。
- 是否有待办。

### 2.4 列表字段

| 字段 | 说明 |
| --- | --- |
| 任务名称 | 点击进入详情 |
| 状态 | 彩色标签 |
| 类型 | 科研、出差等 |
| 级别/优先级 | 重点、紧急等 |
| 负责人 | 头像和姓名 |
| 成员 | 最多展示 3 人，超出折叠 |
| 起止时间 | 计划周期 |
| 进度 | 进度条 |
| 风险 | 延期、冲突、资料缺失 |
| 待办 | 当前用户需处理事项 |

### 2.5 批量操作

- 批量导出。
- 批量归档，只允许已完成任务。
- 批量关注。
- 批量取消关注。

## 3. 新建/编辑任务页

### 3.1 页面结构

采用分步向导：

1. 基本信息。
2. 负责人和成员。
3. 分工草案。
4. 冲突检查。
5. 确认发布。

### 3.2 基本信息步骤

字段：

- 任务名称。
- 任务类型。
- 子类型。
- 任务级别。
- 优先级。
- 所属组织。
- 关联项目。
- 可见性。
- 开始时间。
- 截止时间。
- 任务概述。
- 成果要求。

交互：

- 选择任务类型后加载对应模板。
- 选择休假类型时，系统隐藏项目、成果要求等无关字段。
- 选择隐藏可见性时，必须配置授权范围。
- 截止时间早于开始时间时禁止保存。

### 3.3 人员选择步骤

人员选择器展示：

- 姓名。
- 部门。
- 技能标签。
- 当前状态。
- 当前周期负载。
- 是否跨部门。
- 是否存在冲突。

支持选择方式：

- 按姓名搜索。
- 按组织筛选。
- 按技能筛选。
- 从项目成员中选择。
- 从系统推荐候选人中选择。

### 3.4 分工草案步骤

支持为每个成员创建分工：

- 工作内容。
- 分工类型。
- 起止时间。
- 每日投入。
- 成果要求。

默认逻辑：

- 如果只选择一个成员，自动生成一个同名分工。
- 如果选择项目负责人，可由负责人后续补充分工。
- 如果成员跨部门，分工仍可填写，但正式生效需审批。

### 3.5 冲突检查步骤

展示：

- 人员工时超载。
- 全天任务重叠。
- 休假或出差状态冲突。
- 跨部门审批。
- 隐藏项目授权。

每个冲突提供建议动作：

- 调整人员。
- 调整时间。
- 调整投入。
- 发起协调。
- 保存草稿。

## 4. 任务详情页

### 4.1 顶部信息区

展示：

- 任务名称。
- 状态。
- 类型。
- 负责人。
- 起止时间。
- 当前进度。
- 风险标签。

主操作根据状态变化：

- 草稿：编辑、提交。
- 待协调：查看协调、撤回。
- 待确认：确认发布。
- 进行中：更新进度、提交验收、暂停、变更。
- 待验收：验收通过、退回修改。
- 已完成：归档。
- 已归档：查看档案。

### 4.2 概览页签

内容：

- 任务概述。
- 成果要求。
- 里程碑。
- 成员概览。
- 风险概览。
- 最新动态。

### 4.3 分工页签

展示：

- 分工列表。
- 分工责任人。
- 起止时间。
- 进度。
- 成果。
- 状态。

操作：

- 新增分工。
- 编辑分工。
- 提交成果。
- 确认成果。
- 退回修改。

### 4.4 甘特图页签

展示任务和分工时间线：

- 第一层：任务总周期。
- 第二层：里程碑。
- 第三层：成员分工。

交互：

- 点击分工条打开分工详情。
- 风险分工显示红色标记。
- 已完成分工置灰。

### 4.5 资料页签

展示：

- 必需资料清单。
- 已上传资料。
- 资料版本。
- 阶段成果标记。
- 最终成果标记。

### 4.6 审批页签

展示：

- 跨部门协调。
- 强制安排审批。
- 任务变更审批。
- 隐藏项目授权。

### 4.7 日志页签

展示：

- 状态流转。
- 字段变更。
- 评论。
- 审批。
- 资料上传。
- 验收记录。

## 5. 任务验收页

验收内容：

- 任务目标。
- 任务进度。
- 分工完成情况。
- 必需资料完整性。
- 阶段成果。
- 最终成果。
- 风险处理记录。

验收操作：

- 通过。
- 退回修改。
- 要求补充资料。
- 转交其他验收人。

退回必须填写原因，并生成负责人待办。

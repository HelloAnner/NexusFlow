# 智能工具台页面设计

## 1. 页面清单

- 工具台首页。
- 工具详情页。
- 工具配置页。
- 工具使用记录页。

## 2. 工具台首页

### 2.1 页面目标

让用户快速找到适合当前工作的工具。

### 2.2 页面布局

- 顶部：搜索框、分类筛选、我的常用。
- 中部：工具分类区。
- 右侧：推荐工具。

### 2.3 工具卡片

展示：

- 图标。
- 工具名称。
- 工具分类。
- 简短描述。
- 是否外部工具。
- 最近使用标记。

操作：

- 打开。
- 收藏。
- 查看详情。

## 3. 工具详情页

内容：

- 工具名称。
- 工具描述。
- 适用场景。
- 可见范围。
- 关联任务类型。
- 打开按钮。
- 最近使用记录。

如果从任务详情进入，展示当前上下文：

- 当前任务。
- 可传递资料。
- 可用操作。

## 4. 任务详情中的工具推荐

任务资料页或概览页可展示工具推荐：

- 科研任务推荐论文辅助、专利辅助、资料整理。
- 文职任务推荐 PPT、报告撰写、格式转换。
- 出差任务推荐会议纪要、报告模板。
- 技术支持任务推荐问答助手、需求整理。

推荐规则由配置中心维护。

## 5. 工具配置页

仅管理员或授权配置人员可见。

字段：

- 工具名称。
- 分类。
- 描述。
- 入口类型。
- 入口地址。
- 可见角色。
- 可见组织。
- 关联任务类型。
- 是否首页推荐。
- 是否启用。

校验：

- 外链工具必须填写入口地址。
- 内嵌工具必须配置安全策略。
- 停用工具不再出现在首页和任务详情推荐中。

## 6. 工具使用记录页

筛选：

- 工具名称。
- 使用人。
- 组织。
- 来源页面。
- 关联任务。
- 时间范围。

列表字段：

- 使用时间。
- 使用人。
- 工具。
- 来源页面。
- 关联任务。
- 打开方式。

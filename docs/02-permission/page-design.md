# 权限页面设计

## 1. 页面清单

- 角色管理页。
- 角色详情页。
- 数据范围配置页。
- 隐藏项目授权页。
- 权限审计日志页。

## 2. 角色管理页

### 2.1 页面目标

维护系统角色及其默认操作权限。角色配置需要易读、可审计，避免权限变更后无法解释。

### 2.2 列表字段

| 字段 | 说明 |
| --- | --- |
| 角色名称 | 中心主任、部门主任等 |
| 角色类型 | 管理、执行、系统 |
| 默认数据范围 | 本人、本部门、分管范围等 |
| 可派发范围 | 全员、本部门、项目成员等 |
| 是否可审批 | 是/否 |
| 是否启用 | 是/否 |
| 最近修改人 | 审计使用 |
| 最近修改时间 | 审计使用 |

### 2.3 操作

- 新增角色。
- 复制角色。
- 编辑角色。
- 停用角色。
- 查看授权人员。
- 查看变更记录。

## 3. 角色详情页

### 3.1 页签

- 基础信息。
- 菜单权限。
- 操作权限。
- 数据范围。
- 审批权限。
- 授权人员。
- 变更记录。

### 3.2 操作权限矩阵

用矩阵展示模块和操作：

| 模块 | 查看 | 新建 | 编辑 | 派发 | 审批 | 验收 | 归档 | 导出 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 任务 | 可配置 | 可配置 | 可配置 | 可配置 | 可配置 | 可配置 | 可配置 | 可配置 |
| 人员 | 可配置 | 可配置 | 可配置 | 不适用 | 不适用 | 不适用 | 不适用 | 可配置 |
| 资料 | 可配置 | 可配置 | 可配置 | 不适用 | 不适用 | 不适用 | 可配置 | 可配置 |
| 配置 | 可配置 | 可配置 | 可配置 | 不适用 | 不适用 | 不适用 | 不适用 | 不适用 |

### 3.3 保存校验

- 至少保留一个系统管理员角色。
- 中心主任/书记角色不可删除，只能调整授权人员。
- 配置权限变更需要记录原因。
- 高风险权限开启时提示影响范围。

## 4. 数据范围配置页

### 4.1 页面目标

配置角色在不同模块的数据范围，避免同一角色在任务、人员、资料中范围不一致。

### 4.2 配置维度

- 人员范围。
- 任务范围。
- 项目范围。
- 资料范围。
- 报表范围。
- 甘特图范围。

### 4.3 配置项

| 范围类型 | 说明 |
| --- | --- |
| 本人 | 只看自己 |
| 本部门 | 看主组织所属部门 |
| 分管部门 | 看配置的组织集合 |
| 所属项目 | 看本人参与或负责项目 |
| 分管项目 | 看配置的项目集合 |
| 全中心 | 中心范围内全部 |
| 自定义 | 指定组织、项目、人员 |

## 5. 隐藏项目授权页

### 5.1 页面目标

管理隐藏项目或隐藏任务的显式授权，确保未授权人员完全不可见。

### 5.2 页面结构

- 左侧：隐藏项目列表。
- 右侧：授权对象和授权范围。

### 5.3 授权对象

- 人员。
- 角色。
- 组织。
- 项目成员。

### 5.4 授权范围

- 查看任务。
- 查看资料。
- 编辑任务。
- 参与分工。
- 审批。
- 验收。

### 5.5 规则

- 授权必须有开始时间。
- 可配置结束时间。
- 授权变更即时生效。
- 取消授权后，历史操作日志仍保留。

## 6. 权限审计日志页

### 6.1 筛选项

- 操作人。
- 操作类型。
- 模块。
- 对象。
- 时间范围。
- 风险级别。

### 6.2 列表字段

- 操作时间。
- 操作人。
- 模块。
- 对象名称。
- 操作类型。
- 变更摘要。
- 操作来源。

### 6.3 详情弹窗

展示变更前后 JSON 或结构化字段，支持复制和导出。

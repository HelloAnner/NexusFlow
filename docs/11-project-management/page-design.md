# 项目管理页面设计

## 1. 页面清单

- 项目列表页。
- 项目详情页。
- 项目编辑页。
- 项目成员管理页签。
- 项目任务页签。
- 项目资料页签。
- 项目甘特图页签。
- 隐藏项目授权页。

## 2. 项目列表页

### 2.1 页面目标

让用户按权限查看项目集合，快速进入项目详情、项目任务、项目甘特图和项目资料。

### 2.2 筛选项

- 项目名称或编号。
- 项目类型。
- 项目级别。
- 项目状态。
- 归属组织。
- 项目负责人。
- 是否隐藏项目。
- 风险等级。
- 时间范围。

### 2.3 列表字段

| 字段 | 说明 |
| --- | --- |
| 项目名称 | 点击进入项目详情 |
| 项目编号 | 全局唯一 |
| 类型/级别 | 科研、市场等 |
| 状态 | 筹备、进行中、暂停等 |
| 负责人 | 项目负责人 |
| 成员数 | 当前有效成员数 |
| 任务进度 | 项目下任务汇总 |
| 风险 | 最高风险等级 |
| 资料完整率 | 必需资料完成情况 |

### 2.4 操作

- 新建项目。
- 编辑项目。
- 添加成员。
- 查看甘特图。
- 归档项目。
- 管理授权。

操作按钮按权限展示。

## 3. 项目详情页

### 3.1 顶部信息

- 项目名称。
- 项目编号。
- 项目状态。
- 可见性标识。
- 项目负责人。
- 归属组织。
- 起止日期。
- 风险等级。
- 主操作按钮。

主操作：

- 编辑。
- 新建项目任务。
- 添加成员。
- 管理授权。
- 暂停。
- 完成。
- 归档。

### 3.2 页签

页签：

- 概览。
- 成员。
- 任务。
- 甘特图。
- 资料。
- 风险。
- 授权。
- 日志。

## 4. 概览页签

展示：

- 项目概述。
- 成果要求。
- 关键指标。
- 当前重点任务。
- 高风险事项。
- 近期动态。

关键指标：

- 总任务数。
- 进行中任务数。
- 延期任务数。
- 本周投入工时。
- 成员数。
- 资料完整率。

## 5. 成员页签

### 5.1 列表字段

- 姓名。
- 主组织。
- 项目身份。
- 负责工作。
- 当前任务数。
- 本周负载。
- 加入日期。
- 当前有效。

### 5.2 操作

- 添加成员。
- 修改项目身份。
- 修改负责工作。
- 标记退出项目。
- 查看人员详情。
- 查看该成员项目任务。

### 5.3 规则

- 添加项目成员不直接占用工时。
- 选择外部门成员时提示后续任务派发仍需部门审批。
- 隐藏项目成员列表仅授权人员可见。

## 6. 任务页签

展示项目下任务列表。

筛选：

- 任务状态。
- 任务类型。
- 负责人。
- 成员。
- 风险等级。
- 时间范围。

操作：

- 新建项目任务。
- 查看任务详情。
- 批量查看甘特图。
- 导出有权限的任务清单。

## 7. 甘特图页签

项目甘特图展示：

- 项目周期。
- 项目任务。
- 分工时间线。
- 里程碑。
- 风险和冲突标记。
- 成员负载摘要。

交互：

- 点击任务条进入任务详情。
- 点击成员行进入人员详情。
- 点击冲突标记进入冲突详情。
- 支持按任务、成员、阶段切换。

## 8. 资料页签

展示：

- 项目资料。
- 项目下任务资料。
- 阶段成果。
- 最终成果。
- 归档资料。

操作：

- 上传资料。
- 关联任务资料。
- 标记最终成果。
- 查看版本。
- 下载。

## 9. 授权页签

用于隐藏项目和指定范围项目。

展示：

- 授权对象。
- 授权范围。
- 生效时间。
- 失效时间。
- 授权人。
- 授权原因。

操作：

- 新增授权。
- 修改授权范围。
- 撤销授权。
- 查看授权日志。

## 10. 日志页签

日志类型：

- 项目创建。
- 项目信息变更。
- 成员变更。
- 授权变更。
- 状态变更。
- 归档。

每条日志展示操作人、时间、变更摘要和来源页面。

# 项目管理功能设计

## 1. 功能目标

项目用于承接人员项目归属、项目负责人、项目成员、项目甘特图和项目维度报表。项目不是任务的替代品，而是任务的业务归属、权限边界和统计口径。

目标：

- 支持人员归属到多个项目。
- 支持项目负责人、核心成员、参与成员、支持人员等项目身份。
- 支持项目可见性，包含普通项目、隐藏项目和指定范围项目。
- 支持项目下任务、资料、成员、负载和甘特图联动。
- 支持项目归档后保留历史成员、任务和资料快照。

## 2. 项目类型

默认项目类型：

- 科研项目。
- 市场项目。
- 内部专项。
- 创新工作室项目。
- 临时专项。
- 其他。

科研项目默认级别：

- 公司级。
- 处级。
- 中心级。
- 市场类科研。

项目类型和级别都由配置中心维护。

## 3. 项目字段

### 3.1 基础字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 项目编号 | 自动生成 | 是 | 全局唯一 |
| 项目名称 | 文本 | 是 | 项目展示名称 |
| 项目类型 | 枚举 | 是 | 科研、市场、内部专项等 |
| 项目级别 | 枚举 | 否 | 公司级、处级、中心级等 |
| 归属组织 | 组织 | 是 | 统计和管理口径 |
| 项目负责人 | 人员 | 是 | 对项目任务、成员和资料负责 |
| 分管领导 | 人员 | 否 | 用于项目范围审批和报表 |
| 开始日期 | 日期 | 否 | 项目计划开始 |
| 结束日期 | 日期 | 否 | 项目计划结束 |
| 项目状态 | 枚举 | 是 | 筹备、进行中、暂停、完成、归档 |
| 可见性 | 枚举 | 是 | 普通、隐藏、指定范围 |
| 项目概述 | 长文本 | 否 | 背景、目标、范围 |
| 成果要求 | 长文本 | 否 | 最终交付要求 |

### 3.2 成员字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 人员 | 人员 | 是 | 项目成员 |
| 项目身份 | 枚举 | 是 | 负责人、核心成员、参与成员、支持人员 |
| 负责工作 | 文本 | 否 | 在项目中的职责 |
| 所属组织快照 | 组织快照 | 是 | 加入项目时的主组织 |
| 加入日期 | 日期 | 是 | 成员加入时间 |
| 退出日期 | 日期 | 否 | 成员退出时间 |
| 当前有效 | 布尔 | 是 | 是否仍为有效成员 |

### 3.3 授权字段

隐藏项目授权字段：

- 授权对象：人员、角色或组织。
- 授权范围：查看、参与任务、上传资料、审批、报表。
- 授权原因。
- 授权人。
- 生效时间。
- 失效时间。

## 4. 项目状态

状态流：

```text
筹备 -> 进行中 -> 完成 -> 归档
       |       |
       |       -> 暂停 -> 进行中
       -> 取消
```

状态说明：

- 筹备：项目已创建，但不一定有正式任务。
- 进行中：项目可以创建任务、分配成员、上传资料。
- 暂停：项目暂停，项目下新任务默认不可发布，已有任务按变更规则处理。
- 完成：项目业务完成，可继续整理资料。
- 归档：项目资料和任务快照锁定。
- 取消：项目终止，保留原因和审批记录。

## 5. 项目与任务关系

规则：

- 一个任务最多关联一个主项目。
- 任务可以不关联项目，作为独立工作、个人事项、出差、休假或学习记录。
- 项目负责人可以派发项目相关任务，但不能绕过外部门资源审批。
- 项目成员不是任务成员，只有被加入任务后才产生任务进度和工时占用。
- 项目归档后，项目下未归档任务需先完成、取消或迁移。

## 6. 项目与权限关系

项目范围影响：

- 项目列表。
- 项目详情。
- 项目任务。
- 项目资料。
- 项目甘特图。
- 项目报表。
- 全局搜索结果。

隐藏项目规则：

- 未授权用户不得看到隐藏项目名称、摘要、成员、任务和资料。
- 如果用户因某个隐藏项目任务被授权，只能查看该任务所需的最小项目上下文。
- 隐藏项目的资料下载、报表明细和通知摘要都必须继承隐藏规则。

## 7. 项目成员管理

成员来源：

- 项目负责人手动添加。
- 部门主任推荐。
- 任务派发时因任务成员加入项目候选池。
- 配置导入。

成员变更规则：

- 添加外部门人员到项目成员列表不等于占用资源。
- 真正占用工时必须通过任务或分工产生。
- 外部门成员参与具体任务时仍需协调审批。
- 退出项目后，历史任务和资料仍保留成员当时身份。

## 8. 项目统计

项目统计指标：

- 项目任务数。
- 进行中任务数。
- 延期任务数。
- 高风险任务数。
- 项目成员数。
- 本周预计投入工时。
- 资料归档完整率。
- 阶段成果完成率。

指标口径：

- 只统计当前用户有权限查看的数据。
- 隐藏项目统计明细按授权范围裁剪。
- 项目整体汇总可展示给项目负责人、归属组织管理者和中心领导。

## 9. 项目归档

归档前校验：

- 项目下任务均为已完成、已归档或已取消。
- 必需项目资料已上传。
- 隐藏项目授权仍有归档查看人。
- 未处理高风险冲突为 0。

归档内容：

- 项目基础信息。
- 项目成员历史。
- 项目任务清单。
- 项目资料清单。
- 项目甘特图快照。
- 项目统计快照。
- 项目授权记录。

## 10. 与其他模块联动

- 人员管理：展示人员项目归属、项目身份和负责工作。
- 权限：项目范围是数据权限的重要组成部分。
- 任务管理：任务可关联项目并继承项目可见性。
- 派发协同：项目负责人确认项目任务内容，部门负责人审批资源调用。
- 负载冲突：项目甘特图和成员负载按任务分工计算。
- 资料库：项目资料与任务资料可互相引用。
- 通知报表：项目风险、项目统计、项目资料完整率进入报表。
